Følg trinnene nedenfor.
Velg Start > Control Panel (Kontrollpanel) > Network and Internet Connections (Nettverks- og Internett-tilkoblinger) > Network Connections (Nettverkstilkoblinger).
Høyreklikk ikonet Wireless Network Connection (Trådløs nettverkstilkobling), og velg deretter View Available Networks (Vis tilgjengelige nettverk).
Velg SSID en som vises på skriverens LCD-panel.
