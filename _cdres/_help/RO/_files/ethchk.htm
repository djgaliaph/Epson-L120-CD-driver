Depanarea raportului de verificare al conexiunii

Pentru conexiuni Wi-Fi

Conexiune Ethernet

Rezolvarea problemelor legate de starea reţelei sau de conexiunea la reţea

  1. Consultaţi raportul de conectare la reţea tipărit. Verificaţi elementele din A (Verificarea conexiunii reţelei) pentru care starea indică NEREUŞIT.
  2. În lista cu elemente de verificat, faceţi clic pe cele care prezintă starea NEREUŞIT, pentru a afişa soluţiile.
  3. Verificaţi mesajul afişat în B şi căutaţi acelaşi mesaj listat sub fiecare element de verificat. Apoi urmaţi soluţiile.


Elemente de verificat

Faceţi clic pe articolele de mai jos pentru a ajunge la mesaje şi soluţii.
  1. Ethernet Cable Connection Check (Verificare conexiune cablu Ethernet)
  2. IP Address Check (Verificare adresă IP)
  3. Detailed IP Setup Check (Verificare configurare detaliată IP)


Mesaje şi soluţii pentru elementele de verificat

1. Ethernet Cable Connection Check (Verificare conexiune cablu Ethernet)

Nr. Mesaje Soluţii
1 Verificaţi dacă este conectat cablul de reţea şi dacă dispozitivele de reţea de exemplu hubul, ruterul sau punctul de acces sunt pornite. Verificaţi următoarele:
  • Cablul de reţea este conectat la dispozitivele de reţea, de exemplu la imprimantă sau la un hub
  • Dispozitivele de reţea, de exemplu hubul, sunt pornite
Verificaţi dacă este conectat sau nu cablul de reţea şi dacă acesta funcţionează corect.
Puteţi confirma starea de funcţionare a dispozitivelor de reţea prin verificarea LED-ului de stare asociat portului de reţea de pe dispozitivele conectate, de exemplu de pe hub.

Sus

2. IP Address Check (Verificare adresă IP)

Nr. Mesaje Soluţii
2 Imprimantei dumneavoastră i-a fost alocată o adresă IP incorectă. Verificaţi configurarea adresei IP a dispozitivului de reţea (hub, ruter sau punct de acces). Pentru asistenţă, contactaţi administratorul dumneavoastră de reţea. Verificaţi dacă este activată sau nu funcţia DHCP pentru punctul de acces. Dacă este dezactivată, aceasta trebuie activată.
Sus

3. Detailed IP Setup Check (Verificare configurare detaliată IP)

Nr. Mesaje Soluţii
3 Verificaţi modul de configurare a adresei IP, măştii de subreţea şi adresei gateway implicite.
Conectarea folosind EpsonNet Setup este disponibilă.
Pentru asistenţă, contactaţi administratorul dumneavoastră de reţea.
  1. Verificaţi următoarele:
    • Dacă sunt pornite alte dispozitive din reţea
    • Cablul de reţea este conectat corect
    • Adresa de reţea setată manual pentru imprimantă este corectă
    • Adresa de reţea pentru imprimantă este aceeaşi ca adresa pentru alte dispozitive
  2. Dacă adresa de reţea este incorectă, setaţi-o pe cea corectă, folosind panoul LCD al imprimantei.
    Dacă nu puteţi modifica setările folosind imprimanta, definiţi setările folosind EpsonNet Setup.
    Adresa IP actuală, masca de subreţea şi poarta gateway implicite sunt afişate în C (Network Status (Stare reţea)).
4 Configurarea incompletă. Verificaţi configurare adresei gateway implicite.
Conectarea folosind EpsonNet Setup este disponibilă.
Pentru asistenţă, contactaţi administratorul dumneavoastră de reţea.
  1. Verificaţi următoarele:
    • Adresa implicită a porţii gateway configurată manual pentru imprimantă este corectă
    • Dispozitivul specificat ca gateway implicit este pornit
    • Cablul de reţea este conectat corect
  2. Dacă adresa implicită a gateway-ului este incorectă, setaţi-o pe cea corectă, folosind panoul LCD al imprimantei.
    Dacă nu puteţi modifica setările folosind imprimanta, definiţi setările folosind EpsonNet Setup.
    Adresa implicită actuală a porţii gateway este afişată în C (Network Status (Stare reţea)), sub Default Gateway (Gateway implicit).
5 Verificaţi configurarea conexiunii şi a reţelei calculatorului sau a altui disposzitiv.
Conectarea folosind EpsonNet Setup este disponibilă.
Pentru asistenţă, contactaţi administratorul dumneavoastră de reţea.
  1. Verificaţi următoarele:
    • Dacă sunt pornite alte dispozitive din reţea
    • Cablul de reţea este conectat corect
    • Adresa de reţea setată manual pentru imprimantă este corectă
    • Adresa de reţea pentru imprimantă este aceeaşi ca adresa pentru alte dispozitive
  2. Dacă aţi verificat toate punctele de mai sus, încercaţi următoarele:
    • Porniţi EpsonNet Setup pe calculatorul care foloseşte aceeaşi reţea ca şi imprimanta. Dacă imprimanta apare în lista cu imprimante, setaţi adresa IP a acesteia în ecranul de configurare a adresei IP.
Verificaţi adresa IP a imprimantei şi dacă imprimanta de reţea poate să acceseze sau nu alte dispozitive din reţea. În unele situaţii, eroarea nu poate fi specificată. Se recomandă să efectuaţi configurarea reţelei folosind EpsonNet Setup.
