Führen Sie die folgenden Schritte aus.
Wählen Sie Start > Control Panel (Systemsteuerung) > Network and Internet Connections (Netzwerk- und Internetverbindungen) > Network Connections (Netzwerkverbindungen).
Klicken Sie mit der rechten Maustaste auf das Symbol Wireless Network Connection (Drahtlosnetzwerkverbindung) und wählen Sie View Available Networks (Verfügbare Netzwerke anzeigen).
Wählen Sie die im Drucker-LCD-Bildschirm angezeigte SSID.
