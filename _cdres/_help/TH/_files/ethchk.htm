การแก้ปัญหาสำหรับรายงานการเชื่อมต่อเครือข่าย

สำหรับการเชื่อมต่อ Wi-Fi

การเชื่อมต่ออีเทอร์เน็ต

วิธีการแก้ไขปัญหาสถานะเครือข่ายหรือการเชื่อมต่อ

  1. ดูรายงานการเชื่อมต่อเครือข่ายที่พิมพ์ออกมา ตรวจสอบรายการใน A (Check Network Connection (ตรวจสอบการเชื่อมต่อเครือข่าย)) ที่แสดงสถานะว่า FAIL (ล้มเหลว)
  2. ในลิสต์รายการตรวจสอบ คลิกรายการที่แสดงสถานะ FAIL (ล้มเหลว) เพื่อดูวิธีแก้ไขปัญหา
  3. ตรวจสอบข้อความที่ปรากฏใน B และค้นหาข้อความเดียวกันภายใต้รายการตรวจสอบแต่ละรายการ จากนั้นทำตามวิธีแก้ไขปัญหา


รายการตรวจสอบ

คลิกที่รายการต่างๆ ด้านล่างเพื่อไปยังข้อความและวิธีแก้ไขปัญหา
  1. Ethernet Cable Connection Check (ตรวจสอบการเชื่อมต่อสายเคเบิลอีเทอร์เน็ต)
  2. IP Address Check (ตรวจสอบ IP แอดเดรส)
  3. Detailed IP Setup Check (ตรวจสอบการตั้งค่า IP แบบละเอียด)


ข้อความและวิธีแก้ไขปัญหาของรายการตรวจสอบ

1. Ethernet Cable Connection Check (ตรวจสอบการเชื่อมต่อสายเคเบิลอีเทอร์เน็ต)

เลขที่ ข้อความ วิธีแก้ไขปัญหา
1 Confirm that the network cable is connected and network devices such as hub, router, or access point are turned on. (ตรวจสอบว่าเชื่อมต่อสายเคเบิลเครือข่ายแล้ว และเปิดอุปกรณ์เครือข่าย เช่น ฮับ เราเตอร์ หรือจุดเข้าใช้งานแบบไร้สายแล้ว) ตรวจสอบสิ่งต่างๆ ต่อไปนี้:
  • สายเคเบิลเครือข่ายเชื่อมต่อกับอุปกรณ์เครือข่าย เช่น เครื่องพิมพ์ หรือฮับแล้ว
  • เปิดอุปกรณ์เครือข่าย เช่น ฮับแล้ว
ตรวจสอบว่าสายเคเบิลเครือข่ายเชื่อมต่อและทำงานอย่างถูกต้องหรือไม่
ท่านสามารถยืนยันสถานะการทำงานของอุปกรณ์เครือข่าย โดยตรวจสอบไฟ LED แสดงสถานะของพอร์ตเครือข่ายบนอุปกรณ์ที่เชื่อมต่อ เช่น ฮับ

2. IP Address Check (ตรวจสอบ IP แอดเดรส)

เลขที่ ข้อความ วิธีแก้ไขปัญหา
2 Incorrect IP address is assigned to the printer. (ระบุ IP แอดเดรสที่ไม่ถูกต้องให้กับเครื่องพิมพ์) Confirm IP address setup of the network device (hub, router, or access point). (ตรวจสอบการตั้งค่า IP แอดเดรสของอุปกรณ์เครือข่าย (ฮับ เราเตอร์ หรือจุดเข้าใช้งานแบบไร้สาย)) Contact your network administrator for assistance. (ติดต่อผู้ดูแลเครือข่ายเพื่อขอความช่วยเหลือ) ตรวจสอบว่าฟังก์ชัน DHCP ของจุดเข้าใช้งานแบบไร้สายเปิดใช้งานแล้วหรือไม่ หากไม่ได้เปิดใช้งาน ให้เปลี่ยนการตั้งค่าเป็นเปิดใช้งาน

3. Detailed IP Setup Check (ตรวจสอบการตั้งค่า IP แบบละเอียด)

เลขที่ ข้อความ วิธีแก้ไขปัญหา
3 Confirm IP address, subnet mask, and default gateway setup. (ตรวจสอบการตั้งค่า IP แอดเดรส ซับเน็ตมาสก์ และเกตเวย์เริ่มต้น)
Connection using the EpsonNet Setup is available. (การเชื่อมต่อที่ใช้ EpsonNet Setup สามารถใช้งานได้)
Contact your network administrator for assistance. (ติดต่อผู้ดูแลเครือข่ายเพื่อขอความช่วยเหลือ)
  1. ตรวจสอบสิ่งต่างๆ ต่อไปนี้:
    • อุปกรณ์อื่นๆ บนเครือข่ายเปิดอยู่
    • สายเคเบิลเครือข่ายเชื่อมต่ออย่างถูกต้องแล้ว
    • ที่อยู่เครือข่ายที่ตั้งค่าด้วยตนเองสำหรับเครื่องพิมพ์ถูกต้อง
    • ที่อยู่เครือข่ายสำหรับเครื่องพิมพ์เป็นที่อยู่เดียวกับอุปกรณ์อื่นๆ
  2. หากที่อยู่เครือข่ายไม่ถูกต้อง ให้ตั้งค่าที่อยู่ที่ถูกต้องโดยใช้แผง LCD ของเครื่องพิมพ์
    หากท่านไม่สามารถเปลี่ยนแปลงการตั้งค่าโดยใช้เครื่องพิมพ์ได้ ให้ตั้งค่าโดยใช้ EpsonNet Setup
    IP แอดเดรส ซับเน็ตมาสก์ และเกตเวย์เริ่มต้นปัจจุบันจะแสดงใน C (Network Status (สถานะเครือข่าย))
4 Setup is incomplete. (การตั้งค่าไม่สมบูรณ์) Confirm default gateway setup. (ตรวจสอบการตั้งค่าเกตเวย์เริ่มต้น)
Connection using the EpsonNet Setup is available. (การเชื่อมต่อที่ใช้ EpsonNet Setup สามารถใช้งานได้)
Contact your network administrator for assistance. (ติดต่อผู้ดูแลเครือข่ายเพื่อขอความช่วยเหลือ)
  1. ตรวจสอบสิ่งต่างๆ ต่อไปนี้:
    • ที่อยู่เกตเวย์เริ่มต้นที่ตั้งค่าด้วยตนเองสำหรับเครื่องพิมพ์ถูกต้อง
    • เปิดอุปกรณ์ที่กำหนดเป็นเกตเวย์เริ่มต้นแล้ว
    • สายเคเบิลเครือข่ายเชื่อมต่ออย่างถูกต้องแล้ว
  2. หากที่อยู่เกตเวย์เริ่มต้นไม่ถูกต้อง ให้ตั้งค่าที่อยู่ที่ถูกต้องโดยใช้แผง LCD ของเครื่องพิมพ์
    หากท่านไม่สามารถเปลี่ยนแปลงการตั้งค่าโดยใช้เครื่องพิมพ์ได้ ให้ตั้งค่าโดยใช้ EpsonNet Setup
    ที่อยู่เกตเวย์เริ่มต้นปัจจุบันจะแสดงใน C (Network Status (สถานะเครือข่าย)) ภายใต้เกตเวย์เริ่มต้น
5 Confirm the connection and network setup of the PC or other device. (ตรวจสอบการเชื่อมต่อและการตั้งค่าเครือข่ายของคอมพิวเตอร์หรืออุปกรณ์อื่น)
Connection using the EpsonNet Setup is available. (การเชื่อมต่อที่ใช้ EpsonNet Setup สามารถใช้งานได้)
Contact your network administrator for assistance. (ติดต่อผู้ดูแลเครือข่ายเพื่อขอความช่วยเหลือ)
  1. ตรวจสอบสิ่งต่างๆ ต่อไปนี้:
    • อุปกรณ์อื่นๆ บนเครือข่ายเปิดอยู่
    • สายเคเบิลเครือข่ายเชื่อมต่ออย่างถูกต้องแล้ว
    • ที่อยู่เครือข่ายที่ตั้งค่าด้วยตนเองสำหรับเครื่องพิมพ์ถูกต้อง
    • ที่อยู่เครือข่ายสำหรับเครื่องพิมพ์เป็นที่อยู่เดียวกับอุปกรณ์อื่นๆ
  2. หากท่านตรวจสอบทุกข้อข้างต้นแล้ว โปรดลองดำเนินการดังต่อไปนี้:
    • เริ่มต้น EpsonNet Setup ในคอมพิวเตอร์ที่ใช้เครือข่ายเดียวกับเครื่องพิมพ์ หากเครื่องพิมพ์ของท่านปรากฏในรายชื่อเครื่องพิมพ์ ให้ตั้งค่า IP แอดเดรสของเครื่องพิมพ์บนหน้าจอการตั้งค่า IP แอดเดรส
ตรวจสอบ IP แอดเดรสของเครื่องพิมพ์ และดูว่าเครื่องพิมพ์บนเครือข่ายสามารถเข้าถึงอุปกรณ์อื่นๆ บนเครือข่ายได้หรือไม่ ในบางกรณี จะไม่สามารถระบุข้อผิดพลาดที่เกิดขึ้นได้ ขอแนะนำให้ตั้งค่าเครือข่ายโดยใช้ EpsonNet Setup
