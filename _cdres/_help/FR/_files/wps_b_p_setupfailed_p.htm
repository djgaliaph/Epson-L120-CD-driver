Un écran d’avertissement ou d’erreur s’affiche pour indiquer que la configuration Wi-Fi a échoué.
Appuyez sur OK, vérifiez les éléments suivants, puis procédez de nouveau à la configuration réseau.
Vérifiez que le routeur sans fil (point d’accès) est sous tension. S’il est hors tension, mettez-le sous tension et procédez de nouveau à la configuration réseau.
Si vous avez procédé à la configuration à l’aide d’un code PIN, vérifiez que le code PIN saisi au niveau de l’utilitaire du routeur sans fil (point d’accès) est correct.
Placez l’imprimante à proximité du routeur sans fil (point d’accès).
Vérifiez que le mode de communication Wi-Fi (IEEE802.11*) défini pour le point d’accès correspond à celui pris en charge par l’imprimante (*la valeur affichée varie en fonction du pays).
Si la fonction de filtrage d’adresses MAC est activée, enregistrez l’adresse MAC de votre imprimante afin que l’imprimante ne soit pas bloquée par le filtre.
Vérifiez que la fonction DHCP du point d'accès est bien activée. Si elle est désactivée, activez-la.
