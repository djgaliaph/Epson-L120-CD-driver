Informazioni sulla connettività Internet

Il software EPSON consente la connessione a Internet per i seguenti scopi:
Mantiene il software aggiornato alla versione più recente
Installa le caratteristiche più recenti
Questo software consente di installare le funzioni e le caratteristiche più recenti man mano che si rendono disponibili.
Fare clic sulle caselle sotto riportate per abilitare o disabilitare tali opzioni.
Se appaiono messaggi relativi al firewall durante l'installazione, selezionare [Sblocca] o [Consenti].
