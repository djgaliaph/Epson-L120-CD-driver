Sledite korakom spodaj:
Izberite Start > Control Panel (Nadzorna plošča) > Network and Internet Connections (Omrežne in internetne povezave) > Network Connections (Omrežne povezave).
Z desno tipko miške kliknite ikono Wireless Network Connection (Brezžična omrežna povezava) in nato izberite View Available Networks (Ogled razpoložljivih omrežij).
Izberite SSID, ki je prikazan na zaslonu LCD tiskalnika.
