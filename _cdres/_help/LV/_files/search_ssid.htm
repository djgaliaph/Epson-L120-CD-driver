Izpildiet tālāk norādītās darbības.
Atlasiet iespēju Start (Sākt), Control Panel (Vadības panelis), Network and Internet Connections (Tīkla un interneta savienojumi) > Network Connections (Tīkla savienojumi).
Noklikšķiniet labo peles pogu uz ikonas Wireless Network Connection (Bezvadu tīkla savienojums) un tad atlasiet iespēju View Available Networks (Rādīt pieejamos tīklus).
Atlasiet SSID, kas redzams printera LCD ekrānā.
