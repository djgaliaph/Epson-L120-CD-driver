Feilsøking for rapport om nettverkstilkobling

For Wi-Fi-tilkobling

Ethernet-tilkobling

Slik løser du problemer med nettverksstatusen eller tilkoblingen

  1. Vis rapporten om nettverkstilkobling som du har skrevet ut. Kontroller elementene i A (Kontroller nettverkstilkobling) som har statusen IKKE BESTÅTT.
  2. På listen Kontrollelementer klikker du elementene som viser statusen IKKE BESTÅTT for å vise løsningene.
  3. Kontroller meldingen som vises i B, og se etter samme melding som står oppført under hvert kontrollelement. Følg deretter løsningene.


Kontrollelementer

Klikk elementene nedenfor for å gå til meldingene og løsningene.
  1. Ethernet Cable Connection Check (Kontroll av Ethernet-kabeltilkobling)
  2. IP Address Check (Kontroll av IP-adresse)
  3. Detailed IP Setup Check (Detaljert kontroll av IP-konfigurasjon)


Meldinger og løsninger for kontrollelementene

1. Ethernet Cable Connection Check (Kontroll av Ethernet-kabeltilkobling)

Nr. Meldinger Løsninger
1 Kontroller at nettverkskabelen er tilkoblet og at nettverksenheter, for eksempel hub, ruter eller tilgangspunkt, er slått på. Kontroller følgende:
  • Nettverkskabelen er koblet til nettverksenheter, for eksempel skriveren eller en hub
  • Nettverksenheter, for eksempel en hub, er slått på
Kontroller at nettverkskabelen er tilkoblet og fungerer som den skal.
Du kan bekrefte driftsstatusen til nettverksenheter ved å kontrollere statuslampen for nettverksporten for tilkoblede enheter, for eksempel en hub.

Til toppen

2. IP Address Check (Kontroll av IP-adresse)

Nr. Meldinger Løsninger
2 Feil IP-adresse er tilordnet skriveren. Kontroller konfigurasjon av IP-adresse for nettverksenheten (hub, ruter eller tilgangspunkt). Kontakt systemansvarlig for å få hjelp. Kontroller om DHCP-funksjonen for tilgangspunktet er aktivert. Hvis den er deaktivert, endrer du den til aktivert.
Til toppen

3. Detailed IP Setup Check (Detaljert kontroll av IP-konfigurasjon)

Nr. Meldinger Løsninger
3 Kontroller konfigurasjon av IP-adresse, nettverksmaske og standard gateway.
Tilkobling med EpsonNet-konfigurasjon er tilgjengelig.
Kontakt systemansvarlig for å få hjelp.
  1. Kontroller følgende:
    • Andre enheter på nettverket er slått på
    • Nettverkskabelen er riktig tilkoblet
    • Nettverksadressen som er angitt manuelt for skriveren, er riktig
    • Nettverksadressen for skriveren er den samme som for andre enheter
  2. Hvis nettverksadressen er feil, angir du riktig adresse ved hjelp av skriverens LCD-panel.
    Hvis du ikke kan endre innstillingene med skriveren, foretar du innstillingene med EpsonNet Setup.
    Gjeldende IP-adresse, nettverksmaske og standard gateway vises i C (Network Status (Nettverksstatus)).
4 Konfigurasjonen er ufullstendig. Kontroller konfigurasjon av standard gateway.
Tilkobling med EpsonNet-konfigurasjon er tilgjengelig.
Kontakt systemansvarlig for å få hjelp.
  1. Kontroller følgende:
    • Adressen for standard gateway som er angitt manuelt for skriveren, er riktig
    • Enheten som er angitt som standard gateway, er slått på
    • Nettverkskabelen er riktig tilkoblet
  2. Hvis adressen for standard gateway er feil, angir du riktig adresse ved hjelp av skriverens LCD-panel.
    Hvis du ikke kan endre innstillingene med skriveren, foretar du innstillingene med EpsonNet Setup.
    Gjeldende adresse for standard gateway vises i C (Network Status (Nettverksstatus)) under Standard gateway.
5 Kontroller tilkobling og nettverkskonfigurasjon på PC eller annen enhet.
Tilkobling med EpsonNet-konfigurasjon er tilgjengelig.
Kontakt systemansvarlig for å få hjelp.
  1. Kontroller følgende:
    • Andre enheter på nettverket er slått på
    • Nettverkskabelen er riktig tilkoblet
    • Nettverksadressen som er angitt manuelt for skriveren, er riktig
    • Nettverksadressen for skriveren er den samme som for andre enheter
  2. Hvis du har kontrollert alle punktene ovenfor, prøver du følgende:
    • Start EpsonNet Setup på datamaskinen som bruker samme nettverk som skriveren. Hvis skriveren står oppført på skriverlisten, angir du IP-adressen for skriveren på skjermbildet for konfigurasjon av IP-adresse.
Kontroller skriverens IP-adresse og om nettverksskriveren kan få tilgang til andre enheter på nettverket. Noen ganger hender det at feilen ikke kan angis. Vi anbefaler at du foretar nettverkskonfigurasjonen med EpsonNet Setup.
