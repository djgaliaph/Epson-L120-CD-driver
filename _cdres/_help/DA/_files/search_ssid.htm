Følg nedenstående trin.
Vælg Start > Control Panel (Kontrolpanel) > Network and Internet Connections (Netværks- og internetforbindelser) > Network Connections (Netværksforbindelser).
Højreklik på ikonet Wireless Network Connection (Trådløs netværksforbindelse), og vælg derefter View Available Networks (Vis tilgængelige netværk).
Vælg det SSID, der vises på printerens LCD-skærm.
