Натисніть Start (Пуск) > Control Panel (Панель керування) > Network and Internet Connections (Підключення до мережі та Інтернет) > Network Connections (Мережні підключення).
Правою кнопкою клацніть по значку Wireless Network Connection (Безпроводове мережне підключення), а потім виберіть View Available Networks (Перегляд доступних мереж).
Виберіть SSID, що відображається на рідкокристалічному екрані принтера.
