Postupujte podľa nižšie uvedených krokov.
Vyberte položku Start (Štart) > Control Panel (Ovládací panel) >Network and Internet Connections (Sieť a internet) > Network Connections (Sieťové pripojenia).
Pravým tlačidlom kliknite na ikonu Wireless Network Connection (Pripojenie bezdrôtovej siete) a vyberte možnosť View Available Networks (Zobraziť dostupné siete).
Vyberte SSID zobrazené na obrazovke LCD tlačiarne.
