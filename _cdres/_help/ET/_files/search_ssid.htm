Järgige allpool toodud suuniseid.
Klõpsake Start > Control Panel (Juhtpaneel) > Network and Internet Connections (Võrgu- ja internetiühendused) > Network Connections (Võrguühendused).
Tehke parem hiireklõps ikoonil Wireless Network Connection (Traadita võrguühendus) ja valige View Available Networks (Vaata olemasolevaid võrke).
Valige printeri LCD-ekraanile kuvatud SSID.
