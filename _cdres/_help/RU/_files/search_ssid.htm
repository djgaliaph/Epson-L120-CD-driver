Выполните следующие действия.
Выберите Start (Пуск) > Control Panel (Панель управления) > Network and Internet Connections (Сетевые подключения и Интернет) > Network Connections (Сетевые подключения).
Щелкните правой кнопкой мыши значок Wireless Network Connection (Беспроводное сетевое подключение) и затем выберите View Available Networks (Просмотр доступных сетей).
Выберите имя беспроводной сети (SSID), отображаемое на ЖК-экране принтера.
