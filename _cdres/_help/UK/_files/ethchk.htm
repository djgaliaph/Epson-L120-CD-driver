Для підключення Wi-Fi

Підключення Ethernet

Як вирішити проблеми зі статусом мережі або підключенням

  1. Перегляньте надрукований звіт про мережне підключення. Перевірте пункти у розділі A (Перевірити підключення мережі), для яких відображається статус НЕ ПРОЙДЕНО.
  2. У списку пунктів для перевірки натисніть пункти, для яких відображається статус НЕ ПРОЙДЕНО, щоб подивитися рішення.
  3. Перевірте повідомлення, яке відображається у розділі B і знайдіть це повідомлення під кожним пунктом для перевірки. Далі дотримуйтеся інструкцій.


Пункти для перевірки

Натисніть пункти нижче, щоб перейти до повідомлень та рішень.
  1. Ethernet Cable Connection Check (Перевірити дротове підключення Ethernet)
  2. IP Address Check (Перевірити IP-адресу)
  3. Detailed IP Setup Check (Перевірити детальну установку IP)


Повідомлення та рішення для пунктів для перевірки

1. Ethernet Cable Connection Check (Перевірити дротове підключення Ethernet)

№. Повідомлення Вирішення
1 Підтвердити, що мережний кабель приєднано та мережні пристрої, такі як концентратор, маршрутизатор або точка доступу, ввімкнені. Перевірте такі умови:
  • Мережний кабель приєднано до мережних пристроїв, таких як принтер або концентратор.
  • Мережні пристрої, наприклад, концентратор, увімкнені.
Перевірте, чи приєднаний мережний кабель та чи працює він належним чином.
Робочий статус мережних пристроїв можна перевірити за допомогою світлодіоду мережного порту на приєднаних пристроях, наприклад, на концентраторі.

Угору

2. IP Address Check (Перевірити IP-адресу)

№. Повідомлення Вирішення
2 Принтеру призначена невірна IP-адреса. Перевірте налаштування IP-адреси мережного пристрою (концентратора, маршрутизатора або точки доступу). За допомогою звертайтеся до адміністратора мережі. Перевірте, чи увімкнено функцію DHCP для точки доступу. Якщо її було вимкнено, увімкніть її.
Угору

3. Detailed IP Setup Check (Перевірити детальну установку IP)

№. Повідомлення Вирішення
3 Перевірте настройку IP-адреси, маски підмережі і стандартного шлюзу.
Доступне підключення з використанням EpsonNet Setup.
За допомогою звертайтеся до адміністратора мережі.
  1. Перевірте такі умови:
    • Інші пристрої у мережі ввімкнені.
    • Мережний кабель підключено правильно.
    • Адреса мережі, налаштована для принтера вручну, є правильною.
    • Адреса мережі для принтера не відрізняється від адреси для інших пристроїв.
  2. Якщо адреса мережі неправильна, встановіть правильну адресу за допомогою РК-панелі принтера.
    Якщо не вдається змінити параметри за допомогою принтера, виконайте налаштування за допомогою EpsonNet Setup.
    Поточна IP-адреса, маска підмережі і стандартний шлюз відобразяться у розділі C (Network Status (Статус мережі)).
4 Налаштування не завершено. Перевірте налаштування стандартного шлюзу.
Доступне підключення з використанням EpsonNet Setup.
За допомогою звертайтеся до адміністратора мережі.
  1. Перевірте такі умови:
    • Адреса стандартного шлюзу, налаштована для принтера вручну, є правильною.
    • Пристрій, вказаний як стандартний шлюз, увімкнено.
    • Мережний кабель підключено правильно.
  2. Якщо адреса стандартного шлюзу неправильна, встановіть правильну адресу за допомогою РК-панелі принтера.
    Якщо не вдається змінити параметри за допомогою принтера, виконайте налаштування за допомогою EpsonNet Setup.
    Поточна адреса стандартного шлюзу відображається у розділі C (Network Status (Статус мережі)).
5 Перевірте підключення і налаштування мережі на ПК або іншого пристрою.
Доступне підключення з використанням EpsonNet Setup.
За допомогою звертайтеся до адміністратора мережі.
  1. Перевірте такі умови:
    • Інші пристрої у мережі ввімкнені.
    • Мережний кабель підключено правильно.
    • Адреса мережі, налаштована для принтера вручну, є правильною.
    • Адреса мережі для принтера не відрізняється від адреси для інших пристроїв.
  2. Після перевірки всіх зазначених вище пунктів спробуйте наступні дії:
    • Запустіть EpsonNet Setup на ПК, який використовує таку ж мережу, що і принтер. Якщо принтер з'явиться у списку принтерів, встановіть IP-адресу для принтера на екрані налаштування IP-адреси.
Перевірте IP-адресу принтера та перевірте, чи має мережний принтер доступ до інших пристроїв у мережі. У деяких випадках помилку неможливо визначити. Рекомендуємо виконувати налаштування мережі за допомогою EpsonNet Setup.
