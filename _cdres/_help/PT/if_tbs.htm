Resolução de problemas para uma interface de rede

Descrição geral do problema:

Se pretender ligar a impressora e o computador através de uma rede, o computador tem de ser ligado previamente à rede. Se no computador não estiver nenhuma interface de rede ligada à rede de área local (Wi-Fi ou Ethernet), aparece o ecrã "Ocorreu um erro de interface." Siga estas instruções para resolver o problema.

Passo 1

Em primeiro lugar, siga as instruções indicadas para verificar o estado de rede do computador.
Para Windows 8: seleccione Ambiente de trabalho > atalho Definições > Painel de controlo > Ver estado e tarefas da rede.
Para Windows 7/Vista: Seleccione Start (Iniciar) > Control Panel (Painel de Controlo) > View network status and tasks (Ver estado da rede e tarefas).
Para Windows XP: Seleccione start (iniciar) > Control Panel (Painel de Controlo) > Network and Internet Connections (Ligações de rede e de Internet).
Para Windows 8/7/Vista: clique em Alterar definições de adaptador (para 8/7) ou Gerir ligações de rede (para Vista).
Para Windows XP: Faça clique em Network Connections (Ligações de rede).
Certifique-se de que aparece um ícone de ligação de rede.
Utilizando a tabela seguinte, verifique o ícone de ligação de rede e continue com a solução adequada.

Passo 2

Execute a solução adequada.
Tipo de solução
Estado
Soluções

A
O computador tem o hardware para efectuar ligações de rede Wi-Fi e está activado, mas não está ligado à rede.

Para Windows: O computador tem o hardware para efectuar ligações de rede Wi-Fi, mas não é activado utilizando a tecla ou o interruptor do computador.
・ Se já foi configurada uma rede Wi-Fi utilizando um ponto de acesso, verifique se os dispositivos de rede, como um router ou ponto de acesso, estão ligados. Depois de ligar os dispositivos de rede, tem de aguardar alguns minutos para que a rede funcione correctamente.
・ Se não configurou uma rede Wi-Fi utilizando um ponto de acesso, pode ligar a impressora e o computador através de uma ligação sem fios utilizando o modo Ad Hoc. Pode também utilizar o modo AP simples Wi-Fi Direct se a impressora tiver uma função Wi-Fi Direct. Para mais informações sobre como efectuar uma ligação no modo Ad Hoc ou uma ligação no modo AP simples, consulte "Configuração Ad Hoc no computador" (modo Ad Hoc) ou "Wi-Fi Direct" (modo AP simples) no Guia de Rede.
・ Para Windows: Active a interface de rede. Depois de activar a interface de rede, tem de aguardar alguns minutos para que a rede funcione correctamente.

B
O computador tem o hardware para efectuar ligações de rede Wi-Fi, mas não está activado.
・ Active o estado de rede. Depois de o activar, tem de aguardar alguns minutos para que a rede funcione correctamente.

C
O computador tem o hardware para efectuar ligações de rede Ethernet, mas não está ligado à rede.
・ Se já configurou uma rede Ethernet, verifique se o router ou hub está ligado.
・ Verifique se o cabo LAN está ligado correctamente ao router, hub e computador.

D
O computador tem o hardware para efectuar ligações de rede Ethernet, mas não está activado.
・ Active o estado de rede. Depois de o activar, tem de aguardar alguns minutos para que a rede funcione correctamente.

E
O hardware necessário para ligações Wi-Fi ou Ethernet não existe ou não é reconhecido.
・ Se tiver uma interface de rede instalada ou integrada no computador, certifique-se de que o dispositivo está a funcionar.
・ Se não tiver uma interface de rede instalada ou integrada no computador, não pode utilizar a impressora em rede. Seleccione "USB Wired connection" (Ligação com fios USB) e ligue a impressora e o computador através de um cabo USB. Se pretender utilizar a impressora através de uma rede Wi-Fi ou Ethernet, primeiro configure uma rede para o computador e, em seguida, mude a ligação para a rede. (Consulte o Guia de Rede para mais informações.)
Se não souber configurar o computador ou a interface de rede, consulte as respectivas documentações.

Se o computador estiver ligado directamente à Internet através de uma rede 3G ou outra, sem um router (ou um ponto de acesso) e estiver disponível outra interface de rede, pode efectuar uma ligação sem fios da impressora e do computador utilizando o modo Ad Hoc. Pode também utilizar o modo AP simples Wi-Fi Direct se a impressora tiver uma função Wi-Fi Direct. Contudo, existem restrições e avisos a que deve prestar atenção. Para mais informações sobre como efectuar uma ligação no modo Ad Hoc ou uma ligação no modo AP simples, consulte "Configuração Ad Hoc no computador" (modo Ad Hoc) ou "Wi-Fi Direct" (modo AP simples) no Guia de Rede.
