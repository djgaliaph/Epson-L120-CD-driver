網路連線報告的解決方法

關於乙太網路連線

Wi-Fi 連線

如何解決網路狀態或連線問題

  1. 查看列印出的〝網路連線報告〞。檢查在 A 〝Check Network Connection (檢查網路連線)〞中狀態顯示為〝FAIL (失敗)〞的項目。
  2. 在〝檢查項目〞清單中，點選狀態顯示為〝FAIL (失敗)〞的項目即可顯示解決方法。
  3. 檢查在 B 中所顯示的訊息，並尋找每個〝檢查項目〞下列出的相同訊息。然後請參照解決方法。
關於在 B 中附有 * 的訊息，請參考〝關於網路環境〞的詳細資訊。


檢查項目

點選下列項目，顯示訊息和解決方法。
  1. Wireless Network Name (SSID) Check (無線網路名稱 (SSID) 檢查)
  2. Communication Mode Check (通訊模式檢查)
  3. Security Mode Check (安全模式檢查)
  4. MAC Address Filtering Check (MAC 位址篩選檢查)
  5. Security Key/Password Check (加密金鑰/密碼檢查)
  6. IP Address Check (IP 位址檢查)
  7. Detailed IP Setup Check (詳細的 IP 設定檢查)
    • 關於網路環境


檢查項目的訊息與解決方法

1. Wireless Network Name (SSID) Check (無線網路名稱 (SSID) 檢查)

編號 訊息 解決方法
1 No wireless network names (SSID) found. Confirm that the router/access point is turned on and the wireless network (SSID) is set up correctly. Contact your network administrator for assistance.
  • 檢查是否已開啟無線基地台 (AP) 的電源。如果未開啟電源，請開啟電源，然後再執行網路設定。
  • 將印表機放在靠近無線基地台 (AP) 的位置。
  • 如果您已手動輸入無線網路名稱 (SSID)，請確認您的無線網路名稱 (SSID) 是否正確。如果不正確，請再次輸入。您的無線網路名稱 (SSID) 會顯示在 C 〝Network Status (網路狀態)〞的〝Network Name (SSID) (網路名稱 (SSID))〞中。
  • 如果您已使用 WPS 設定網路，請檢查您的無線基地台 (AP) 是否有支援 WPS。
  • 如果您要設定 Ad Hoc 連線，您必須先在電腦端設定 Ad Hoc。執行完電腦的 Ad Hoc 設定後，再執行印表機的網路設定。
2 No wireless network names (SSID) found. Confirm that the wireless network name (SSID) of the PC you wish to connect is set up correctly. Contact your network administrator for assistance.
  • 檢查是否已開啟您要連接的電腦電源。如果未開啟電源，請開啟電源，然後再執行網路設定。
  • 將印表機放在靠近您要連接的電腦附近。
當在網路設定過程中找不到選取或輸入的無線網路名稱 (SSID) 時，便會顯示上述訊息。
SSID 是 Wi-Fi 網路的名稱。如果您是使用無線基地台 (AP)，那麼當設定了無線基地台 (AP) 時，您的網路便已有名稱。
如果您不透過無線基地台(AP) 直接連接印表機和電腦 (Ad Hoc 連線)，您必須在電腦端設定 Ad Hoc 網路。在這種情況下，您需要在電腦上建立 SSID。
如果您不知道您的 SSID，我們建議您與為您設定網路的人員或公司聯絡。
此外，在設定過程中，您還需要將印表機放在靠近無線基地台 (或電腦) 的位置，如此印表機才能夠從無線基地台 (AP) 或電腦接收到清晰的訊號。

2. Communication Mode Check (通訊模式檢查)

編號 訊息 解決方法
3 Wi-Fi communication mode (IEEE802.11*) does not match the current setting of the printer. Set the same communication mode to the printer, router/access point, and PC. Contact your network administrator for assistance.

*該處的顯示內容會因您的所在地區而不同。
  • 請檢查您的無線基地台 (AP) 所設定的 Wi-Fi 通訊模式與印表機所支援的模式相符。
當印表機不支援無線基地台 (AP) 和電腦所設定的 Wi-Fi 通訊模式時，便會顯示此訊息。

3. Security Mode Check (安全模式檢查)

編號 訊息 解決方法
4 Security mode (e.g. WEP, WPA) does not match the current setting of the printer. Confirm security mode. Contact your network administrator for assistance. 請確認無線基地台 (AP) 是使用下列其中一種安全模式。若未使用，請再執行網路設定並更改為有支援的安全模式。
  • WEP-64bit (40bit)
  • WEP-128bit (104bit)
  • WPA PSK (TKIP)
  • WPA2 PSK (TKIP)
  • WPA PSK (AES)
  • WPA2 PSK (AES)
  • WPA (TKIP/AES)
  • WPA2 (TKIP/AES)
當印表機不支援網路裝置所設定的安全模式，或該模式與印表機的安全模式不符時，便會顯示此訊息。
根據網路裝置的強度或加密方法而定，網路裝置有不同類型的安全設定。
根據網路裝置而定，支援的安全模式可能不同。若要允許通訊，必須在網路裝置與印表機之間設定相同的安全模式。

4. MAC Address Filtering Check (MAC 位址篩選檢查)

編號 訊息 解決方法
5 MAC address of your printer may be filtered. Check whether your router/access point has restrictions such as MAC address filtering.
See the documentation of the router/access point or contact your network administrator for assistance.
  • 關於 MAC 位址篩選功能的詳細資訊，請參考無線基地台 (AP) 內附的操作手冊。若已啟用篩選功能，請註冊印表機的 MAC 位址，以防印表機遭到篩除。
  • 如果您已使用無線基地台 (AP) 的 WEP 安全模式來啟用共用驗證，請確認驗證金鑰與索引皆正確。
當已啟用 MAC 位址篩選，且尚未將印表機的 MAC 位址註冊至無線基地台 (AP) 時，會顯示此訊息。
MAC 位址是用來識別個別網路裝置的識別數字。
無線基地台 (AP) 有一項功能，可允許或拒絕裝置依據其 MAC 位址來存取網路。
如果啟用此功能，只有已註冊 MAC 位址的網路裝置才能連接網路。

5. Security Key/Password Check (加密金鑰/密碼檢查)

編號 訊息 解決方法
6 Entered security key/password does not match the one set for your router/access point. Confirm security key/password. Contact your network administrator for assistance. 無線基地台 (AP) 所設定的加密金鑰與輸入的加密金鑰不符。請使用正確的加密金鑰重新設定網路。加密金鑰有大小寫之分。
當輸入的加密金鑰與無線基地台 (AP) 所設定的金鑰不符時，便會顯示此訊息。
若要連接網路，必須要有加密金鑰 (密碼)。
加密金鑰有大小寫之分。如果輸入的金鑰與註冊的金鑰不完全相符，連線便會失敗。

6. IP Address Check (IP 位址檢查)

編號 訊息 解決方法
7 Incorrect IP address is assigned to the printer. Confirm IP address setup of the network device (hub, router, or access point). Contact your network administrator for assistance.
  • 檢查是否已開啟無線基地台 (AP) 的 DHCP 功能。 若沒有開啟，請開啟此功能。
  • 如果您已手動設定 IP 位址，該 IP 位址不在可支援的範圍內 (無效)。請輸入正確的 IP 位址。

    您目前的預設閘道位址會顯示在 C 〝Network Status (網路狀態)〞的〝Default Gateway (預設閘道)〞中。

-Entered security key/password is correct

    • 印表機的網路位址與其他裝置的網路位址相同

    • 您可以在 WEP 安全模式下為無線基地台 (AP) 註冊多個加密金鑰。 若已註冊多個金鑰，請檢查您是否已輸入第一個註冊的金鑰。
12 Confirm the following:
-Entered security key/password is correct
-Index of the security key/password is set to the first number
-Connection and network setup of the PC or other device is correct
Contact your network administrator for assistance.
  1. 檢查下列項目：
    • 已開啟網路裝置 (例如數據機、集線器、和路由器) 的電源。
    • 印表機以外的網路裝置的 IP 位址是手動設定的 (如果印表機的 IP 位址是自動指定的，但其他網路裝置是手動指定的，印表機的網路可能會與其他網路裝置不同)

    • 在與印表機使用相同網路的電腦上啟動 EpsonNet Setup。

    • 您可以在 WEP 安全模式下為無線基地台 (AP) 註冊多個加密金鑰。 若已註冊多個金鑰，請檢查您是否已輸入第一個註冊的金鑰。
檢查印表機的 IP 位址，並檢查網路印表機是否能夠存取網路上的其他裝置。在某些情況下，可能會無法指出錯誤。 我們建議您使用 EpsonNet Setup 執行網路設定。

關於網路位址：
IP 位址不僅指示連接至網路的裝置 (例如電腦、集線器與路由器)，還有它們所屬的網路範圍。相同網路上的裝置可以互相通訊。如果您想連接至不同網路的裝置，則需要有路由器。IP 位址由兩部分組成。第一部分稱為〝網路位址〞，它指示網路範圍。第二部分稱為〝主機位址〞，它指示網路裝置。網路位址與主機位址之間的分隔線會因網路的規模 (等級) 而不同。但是您可以使用子網路遮罩定義網路位址與主機位址之間的分隔線。
如果您要在家用網路上使用印表機，當使用無線基地台 (AP) 的 DHCP 功能時，請將印表機的 IP 位址設定為〝自動〞。如果您要使用〝手動〞設定印表機的 IP 位址，請參考下列說明進行 IP 位址和子網路遮罩的設定。
  1. 檢視連接至電腦的路由器 (或無線基地台 (AP)) 的 IP 位址。詳細說明，請參考路由器 (或無線基地台 (AP)) 的操作手冊。
  2. 所有 IP 位址都是採用四組數值的格式 (針對 IPv4)。在最後一組數值中，設定介於 1 至 254 之間的唯一數值。(當路由器為 DHCP 伺服器或家用網路中的其他 DHCP 伺服器時，請設定一個不會與 DHCP 伺服器所配置的 IP 位址範圍相衝突的值。)
設定範例：
路由器 (或無線基地台 (AP)) 的 IP 位址：192.168.1.1
印表機的範例設定：192.168.1.3
如果您在小型網路中使用印表機，則所有網路裝置的子網路遮罩可以設為 255.255.255.0。此外，閘道應使用與無線基地台 (AP) 的 IP 位址相同的值。

關於網路環境

與網路狀態/連線檢查報告一樣，也會顯示下列訊息來讓您知道網路環境中有其他哪些方面需要改善。
訊息 解決方法
*Signal strength is low. If you have problems printing or scanning, improve your wireless network environment. 由於訊號強度偏低，列印速度可能較慢或列印可能中斷。
請執行下列項目，改善環境：
  • 將印表機放在靠近無線基地台 (AP) 的位置。
  • 移除無線基地台 (AP) 和印表機之間的所有障礙物。
*A router/access point channel conflict has been detected. If you have problems printing or scanning, improve your wireless network environment. 附近的房間或建築物內所使用的其他無線網路可能會造成訊號干擾。將印表機放在靠近無線基地台 (AP) 的位置可以改善環境。
*Signal strength is low. A router/access point channel conflict has been detected. If you have problems printing or scanning, improve your wireless network environment. 由於訊號強度偏低，列印速度可能較慢或列印可能中斷。附近的房間或建築物內所使用的其他無線網路可能會造成訊號干擾。
請執行下列項目，改善環境：
  • 將印表機放在靠近無線基地台 (AP) 的位置。
  • 移除無線基地台 (AP) 和印表機之間的所有障礙物。
*Multiple network names (SSID) that match your entered network name (SSID) have been detected. Confirm network name (SSID). 您正連接至其他的無線基地台 (AP)。
嘗試下列各項：
變更您要連接的無線基地台 (AP) 的 SSID，然後再執行印表機的網路設定。
*No more devices can be connected. Disconnect one of the connected devices if you want to add another one. 如果您想新增裝置，請使用裝置的 Wi-Fi 設定中斷任一台目前裝置的連線，或從印表機變更密碼以中斷所有裝置的連線。
