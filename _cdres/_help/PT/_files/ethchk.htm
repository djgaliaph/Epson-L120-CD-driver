Resolução de problemas para Relatório de ligação de rede

Para ligação Wi-Fi

Ligação Ethernet

Como resolver problemas de estado ou ligação da rede

  1. Veja o Relatório de ligação de rede que imprimiu. Verifique os itens em A (Verificar Ligação de Rede) cujo estado tem a indicação NÃO APROVADO.
  2. Na lista Verificar itens, faça clique nos itens que apresentam um estado NÃO APROVADO para visualizar as soluções.
  3. Verifique a mensagem apresentada em B e procure a mesma mensagem que aparece por baixo de cada Verificar item. Siga as soluções.


Verificar itens

Faça clique nos itens seguintes para avançar para as mensagens e soluções.
  1. Ethernet Cable Connection Check (Verificar Ligação do Cabo Ethernet)
  2. IP Address Check (Verificar Endereço IP)
  3. Detailed IP Setup Check (Verificar Configuração IP Detalhada)


Mensagens e soluções para os itens de verificação

1. Ethernet Cable Connection Check (Verificar Ligação do Cabo Ethernet)

N.º Mensagens Soluções
1 Confirme se o cabo de rede está ligado e se os dispositivos de rede, como o hub, router ou ponto de acesso estão ligados. Verifique o seguinte:
  • O cabo de rede está ligado a dispositivos de rede, como a impressora ou um hub
  • Os dispositivos de rede, como um hub, estão ligados
Verifique se o cabo de rede está ou não ligado e a funcionar correctamente.
Pode confirmar o estado de funcionamento dos dispositivos de rede verificando o LED de estado da porta de rede nos dispositivos ligados, como um hub.

2. IP Address Check (Verificar Endereço IP)

N.º Mensagens Soluções
2 Endereço IP incorrecto atribuído à impressora. Confirme a configuração do endereço IP do dispositivo de rede (hub, router ou ponto de acesso). Para obter assistência, contacte o seu administrador de rede. Verifique se a função DHCP para o ponto de acesso está ou não activada. Se estiver desactivada, active-a.

3. Detailed IP Setup Check (Verificar Configuração IP Detalhada)

N.º Mensagens Soluções
3 Confirme configuração do endereço IP, máscara de sub-rede e gateway predefinido.
Está disponível uma ligação através do EpsonNet Setup.
Para obter assistência, contacte o seu administrador de rede.
  1. Verifique o seguinte:
    • Outros dispositivos estão ligados na rede
    • O cabo de rede está ligado correctamente
    • O endereço de rede definido manualmente para a impressora está correcto
    • O endereço de rede da impressora é igual ao dos outros dispositivos
  2. Se o endereço de rede estiver incorrecto, defina o endereço correcto utilizando o painel LCD da impressora.
    Se não conseguir alterar as definições através da impressora, efectue as definições utilizando o EpsonNet Setup.
    O seu endereço IP, máscara de sub-rede e gateway predefinido actuais são apresentados em C (Network Status (Estado da Rede)).
4 Configuração incompleta. Confirme configuração do gateway predefinido.
Está disponível uma ligação através do EpsonNet Setup.
Para obter assistência, contacte o seu administrador de rede.
  1. Verifique o seguinte:
    • O endereço do gateway predefinido definido manualmente para a impressora está correcto
    • O dispositivo especificado como gateway predefinido está ligado
    • O cabo de rede está ligado correctamente
  2. Se o endereço do gateway predefinido estiver incorrecto, defina o endereço correcto utilizando o painel LCD da impressora.
    Se não conseguir alterar as definições através da impressora, efectue as definições utilizando o EpsonNet Setup.
    O seu endereço do gateway predefinido actual é apresentado em C (Network Status (Estado da Rede)) por baixo de Gateway predefinido.
5 Confirme a ligação e configuração de rede do PC ou de outro dispositivo.
Está disponível uma ligação através do EpsonNet Setup.
Para obter assistência, contacte o seu administrador de rede.
  1. Verifique o seguinte:
    • Outros dispositivos estão ligados na rede
    • O cabo de rede está ligado correctamente
    • O endereço de rede definido manualmente para a impressora está correcto
    • O endereço de rede da impressora é igual ao dos outros dispositivos
  2. Se verificou todos os pontos anteriores, experimente o seguinte:
    • Inicie o EpsonNet Setup no computador que está a utilizar a mesma rede que a impressora. Se a impressora aparecer na lista de impressoras, defina o endereço IP da impressora no ecrã de configuração do endereço IP.
Verifique o endereço IP da impressora e também se a impressora de rede consegue ou não aceder a outros dispositivos na rede. Em alguns casos, não é possível especificar o erro. Recomendamos que efectue a configuração da rede utilizando o EpsonNet Setup.
