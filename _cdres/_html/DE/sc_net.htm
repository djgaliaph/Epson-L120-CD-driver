Informationen zur Internet-Konnektivität

EPSON-Software kann für folgende Zwecke eine Verbindung mit dem Internet herstellen:
Hält die Software immer auf dem Stand der neuesten Version
Installiert die neuesten Funktionen
Diese Software ermöglicht die Installation der neuesten Funktionen sobald sie verfügbar sind.
Aktivieren Sie die Kontrollkästchen unten, um diese Optionen zu aktivieren oder zu deaktivieren.
Wenn während der Installation Meldungen der Firewall angezeigt werden, wählen Sie [Freigeben] oder [Zulassen].
