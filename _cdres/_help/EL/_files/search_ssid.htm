Ακολουθήστε τα παρακάτω βήματα.
Επιλέξτε Start (Έναρξη) > Control Panel (Πίνακας ελέγχου) > Network and Internet Connections (Συνδέσεις δικτύου και Internet) > Network Connections (Συνδέσεις δικτύου).
Κάντε δεξί κλικ στο εικονίδιο Wireless Network Connection (Σύνδεση ασύρματου δικτύου) και, στη συνέχεια, επιλέξτε View Available Networks (Προβολή διαθέσιμων δικτύων).
Επιλέξτε το SSID που εμφανίζεται στην οθόνη LCD του εκτυπωτή.
