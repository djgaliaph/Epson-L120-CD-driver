网络连接检查报告的故障排除

对于以太网连接

Wi-Fi 连接

如何解决网络状态或连接问题

  1. 查看您所打印的网络连接检查报告。 检查 A（Check Network Connection（检查网络连接））中状态显示为“FAIL（失败）”的项目。
  2. 在“检查项目”列表中，单击状态显示为“FAIL（失败）”的项目，显示其解决方法。
  3. 检查 B 中显示的信息，并在各个“检查项目”下查找列出的相同信息。 然后按照解决方法进行操作。
对于 B 中带有 * 的信息，请参见“关于网络环境”以获得更多信息。


检查项目

单击下方的项目，查看信息和解决方法。
  1. Wireless Network Name (SSID) Check（检查无线网络名称 (SSID)）
  2. Communication Mode Check（检查通讯模式）

检查项目的信息和解决方法

1. Wireless Network Name (SSID) Check（检查无线网络名称 (SSID)）

编号 信息 解决方法
1 No wireless network names (SSID) found.（找不到无线网络名称 (SSID)。） Confirm that the router/access point is turned on and the wireless network (SSID) is set up correctly.（请确认路由器/接入点已开启，并且无线网络名称 (SSID) 设置正确。） Contact your network administrator for assistance.（请联系您的网络管理员以获得帮助。）
  • 检查接入点是否已开启。 若已关闭，请将它开启，然后重新执行网络设置。
  • 将您的打印机置于靠近接入点的位置。
  • 如果手动输入了无线网络名称(SSID)，请确认您的无线网络名称 (SSID)是否正确。若不正确，请重新输入。在C（Network Status（网络状态））中“网络名称 (SSID)”下显示您当前的无线网络名称(SSID)。
  • 如果将网络设为使用WPS，请检查您的接入点是否支持WPS。
  • 如果要设置Ad Hoc连接，需要先为您的计算机设置Ad Hoc。 为计算机设置Ad Hoc，然后重新为打印机执行网络设置。
2 No wireless network names (SSID) found.（找不到无线网络名称 (SSID)。）Confirm that the wireless network name (SSID) of the PC you wish to connect is set up correctly.（请确认您要连接的计算机的无线网络名称 (SSID) 设置正确。） Contact your network administrator for assistance.（请联系您的网络管理员以获得帮助。）
  • 检查您要连接的计算机是否已开启。 若已关闭，请将它开启，然后重新执行打印机的网络设置。
  • 将打印机置于靠近要连接的计算机的位置。
如果网络设置过程中没有找到选定的或输入的无线网络名称 (SSID)，就会显示上述信息。
SSID 是您的 Wi-Fi 网络的名称。 如果您用的是接入点，则您的网络在设置接入点时就已有了名称。
如果要将打印机与计算机不通过接入点直接相连（Ad Hoc 连接），则需要为计算机设置 Ad Hoc 网络。 在这种情形下，您需要在计算机上创建一个 SSID。
如果不知道您的 SSID，建议您与设置网络的人员或公司联系。
另外，在设置时需要将打印机置于靠近接入点（或计算机）的位置，以便打印机能从接入点或计算机接收清晰的信号。

3. Security Mode Check（检查安全性模式）

编号 信息 解决方法
4 Security mode (e.g. WEP, WPA) does not match the current setting of the printer.（安全性模式（如 WEP 和 WPA）与打印机的当前设置不匹配。）Confirm security mode.（请确认安全性模式。） Contact your network administrator for assistance.（请联系您的网络管理员以获得帮助。） 确认接入点使用下列安全性模式之一。 若不是，请重新执行网络设置，将它更改为支持的安全性模式。
  • WEP-64bit (40bit)
  • WEP-128bit (104bit)
  • WPA PSK (TKIP)
  • WPA2 PSK (TKIP)
  • WPA PSK (AES)
  • WPA2 PSK (AES)
  • WPA (TKIP/AES)
  • WPA2 (TKIP/AES)
如果网络设备的安全性模式设置不受打印机支持，或与打印机的安全性模式不匹配，就会显示此信息。
网络设备有几种不同的安全性设置，具体取决于它们的强度或加密方法。
受支持的安全性模式可能因网络设备而异。 要允许通讯，网络设备和打印机之间必须设置相同的安全性模式。

4. MAC Address Filtering Check（检查 MAC 地址过滤）

编号 信息 解决方法
5 MAC address of your printer may be filtered.（打印机的 MAC 地址可能已被过滤。 ）Check whether your router/access point has restrictions such as MAC address filtering.（请检查路由器/接入点是否有 MAC 地址过滤等限制。）
See the documentation of the router/access point or contact your network administrator for assistance.（请参阅路由器/接入点的文档或联系您的网络管理员以获得帮助。）
  • 参阅您的接入点随附的说明手册，获得有关MAC地址过滤功能的更多信息。 如果已开启过滤功能，请注册打印机的MAC地址，以避免打印机被过滤掉。
  • 如果您的接入点使用 WEP 安全性模式来启用共享身份验证，请确认身份验证的密钥和索引均正确。
若已启用 MAC 地址过滤但打印机的 MAC 地址尚未注册到接入点，就会显示此信息。
MAC 地址是用来识别各个网络设备的身份号码。
接入点具有可根据设备的 MAC 地址来决定是否允许该设备访问其网络的功能。
启用此功能时，只有已注册 MAC 地址的网络设备才能连接到网络。

5. Security Key/Password Check（检查安全密钥/密码）

编号 信息 解决方法
6 Entered security key/password does not match the one set for your router/access point.（输入的安全密钥/密码与路由器/接入点的不匹配。）Confirm security key/password.（请确认安全密钥/密码。） Contact your network administrator for assistance.（请联系您的网络管理员以获得帮助。） 为接入点所设的安全密钥与输入的安全密钥不匹配。 请使用正确的安全密钥重新设置您的网络。 安全密钥区分大小写。
如果输入的安全密钥与接入点的安全密钥不匹配，就会显示此信息。
安全密钥（密码）是连接网络的必要条件。
安全密钥区分大小写。 如果输入的密钥与注册的密钥不完全匹配，连接就会失败。

6. IP Address Check（检查 IP 地址）

编号 信息 解决方法
7 Incorrect IP address is assigned to the printer.（打印机分配了不正确的 IP 地址。） Confirm IP address setup of the network device (hub, router, or access point).（请确认网络设备（集线器、路由器或接入点）的 IP 地址设置。） Contact your network administrator for assistance.（请联系您的网络管理员以获得帮助。）
  • 检查是否已启用接入点的DHCP功能。 若未启用，将它改为启用。
  • 如果您手动设置了IP地址，IP地址超出了支持的范围（无效）。 请输入正确的IP地址。

关于网络地址：
IP 地址不仅表示设备已连接到网络（如计算机、集线器和路由器），还表示它们所属的网络范围。 相同网络上的设备可互相通讯。 如果您要在不同网络上的设备间进行通讯，则需要路由器。 IP 地址由两个部分组成。 第一部分称为“网络地址”，它表示的是网络范围。 第二部分称为“主机地址”，它表示的是网络设备。 网络地址与主机地址之间的连线因网络的规模（等级）而异。 但是，您可以使用子网掩码来定义网络地址与主机地址间的连线。
如果您在家庭网络上使用打印机，在使用接入点的DHCP 功能时，可将打印机的 IP 地址设为“自动”。 如果想要使用“手动”来设置打印机的 IP 地址，请参照以下内容来设置 IP 地址和子网掩码。
  1. 检查计算机所连路由器（或接入点）的 IP 地址。 有关详细信息，请参见路由器（或接入点）的说明手册。
  2. 所有 IP 地址均采用四个数值片段的形式书写（对于 IPv4）。 在最后一个片段中，请设置 1 到 254 范围内的一个唯一数字。 （如果路由器为 DHCP 服务器，或者家庭网络中有另外一个 DHCP 服务器，设置的值不能与 DHCP 服务器分配的 IP 地址产生冲突。
设置示例：
路由器（或接入点）IP地址： 192.168.1.1
打印机样例设置： 192.168.1.3
如果您正在小规格的网络上使用您的打印机。设置255.255.255.0作为所有网络设备的子网掩码。 另外，网关使用与接入点 IP 地址相同的值。

关于网络环境

除了网络状态/连接检查报告外，还会显示下列信息，帮助您了解网络中还有哪些因素需要改善。
信息 解决方法
*Signal strength is low.（*信号强度较弱。）If you have problems printing or scanning, improve your wireless network environment.（如果您有打印或扫描的问题，请改善您的无线网络环境。） 由于信号强度比较弱，打印速度可能会缓慢，或打印中断。
执行下列项目以改善环境：
  • 将打印机置于靠近接入点的位置
  • 移除接入点与打印机之间的障碍物。
*A router/access point channel conflict has been detected.（*检测到路由器/接入点频道冲突。）If you have problems printing or scanning, improve your wireless network environment.（如果您有打印或扫描的问题，请改善您的无线网络环境。） 附近房间或楼宇使用的其他无线网络可能会造成信号干扰。 将打印机置于靠近接入点的位置可改善环境。
*Signal strength is low.（*信号强度较弱。）A router/access point channel conflict has been detected.（检测到路由器/接入点频道冲突。）If you have problems printing or scanning, improve your wireless network environment.（如果您有打印或扫描的问题，请改善您的无线网络环境。） 由于信号强度比较弱，打印速度可能会缓慢，或打印中断。 附近房间或楼宇使用的其他无线网络可能会造成信号干扰。

更改要连接的接入点的 SSID，然后重新执行打印机的网络设置。
