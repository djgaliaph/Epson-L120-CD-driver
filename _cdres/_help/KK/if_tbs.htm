Желі интерфейсінің ақаулықтарын жою

Ақаулықты шолу:

Егер принтерді және компьютеріңізді желі арқылы қосқыңыз келсе, компьютеріңізді алдын ала желіге қосу керек. Егер ешқандай желі интерфейсі компьютеріңіздегі жергілікті аймақтағы желіге (Wi-Fi немесе Ethernet) қосылмаған болса, «Интерфейс қатесі пайда болды.» экраны көрсетіледі. Ақаулықты шешу үшін төмендегі қадамдарды орындаңыз.

1-ші қадам

Алдымен, компьютеріңіздің желі күйін тексеру үшін төмендегі қадамдарды орындаңыз.
Windows 8 үшін: Жұмыс үстелі > Зерделі Параметрлер > Басқару тақтасы > Желі күйін және тапсырмаларды көру тармағына өтіңіз.
Windows 7/Vista үшін: Start (Бастау) > Control Panel (Басқару панелі) > View network status and tasks (Желі күйін және тапсырмаларды көру) опциясын таңдаңыз.
Windows XP үшін: start (Бастау) > Control Panel (Басқару панелі) > Network and Internet Connections (Желі және Интернет қосылымдары) опциясын таңдаңыз.
Windows 8/7/Vista үшін: Адаптер параметрлерін өзгерту (8/7 үшін) немесе Желі байланыстарын реттеу (Vista үшін) опциясын таңдаңыз.
Windows XP үшін: Network Connections (Желі қосылымдары) опциясын таңдаңыз.
Желі қосылымының белгішесі көрсетілгеніне көз жеткізіңіз.
Төмендегі кесте арқылы желі қосылымы белгішесін тексеріп, тиісті шешіммен жалғастырыңыз.

A
Компьютерде Wi-Fi желі қосылымдарын орнатуға арналған жабдық бар және ол қосылған, бірақ ол желіге қосылмаған.

Windows үшін: компьютерде Wi-Fi желі қосылымдарын орнатуға арналған жабдық бар, бірақ ол компьютердегі түйме немесе қосқыш арқылы қосылмаған.
・ Егер Wi-Fi желісі кіру нүктесі арқылы бұрыннан орнатылған болса, жол жоспарлағыш немесе кіру нүктесі сияқты желі құрылғылары қосылғанын тексеріңіз. Желі құрылғыларын қосқаннан кейін, желі дұрыс жұмыс істеуінен бұрын бірнеше минут күтіңіз.
・ Егер Wi-Fi желісін кіру нүктесі арқылы орнатпасаңыз, принтер мен компьютеріңізді сымсыз Ad Hoc режимі арқылы қосуға болады. Егер принтеріңізде Wi-Fi Direct функциясы болса, Wi-Fi Direct Simple AP режимін пайдалануға болады. Ad Hoc режимі қосылымын немесе Simple AP режимі қосылымын орнату туралы қосымша мәлімет алу үшін Желі нұсқаулығындағы «Компьютердегі Ad Hoc орнату» (Ad Hoc режимі) немесе «Wi-Fi Direct» (Simple AP режимі) бөлімін қараңыз.
・ Windows үшін: желі интерфейсін қосыңыз. Желі интерфейсін қосқаннан кейін, желі дұрыс жұмыс істеуінен бұрын бірнеше минут күтіңіз.

B
Компьютерде Wi-Fi желі қосылымдарын орнатуға арналған жабдық бар, бірақ ол қосылмаған.
・ Желі күйін іске қосыңыз. Желі күйін іске қосқаннан кейін, желі дұрыс жұмыс істеуінен бұрын бірнеше минут күтіңіз.

C
Компьютерде Ethernet желі қосылымдарын орнатуға арналған жабдық бар, бірақ ол желіге қосылмаған.
・ Егер Ethernet желісі бұрыннан орнатылған болса, жол жоспарлағыш немесе хабтың қосылғанын тексеріңіз.
・ Жергілікті желі кабелі жол жоспарлағышқа, хабқа және компьютерге дұрыс жалғанғанын тексеріңіз.

D
Компьютерде Ethernet желі қосылымдарын орнатуға арналған жабдық бар, бірақ ол қосылмаған.
・ Желі күйін іске қосыңыз. Желі күйін іске қосқаннан кейін, желі дұрыс жұмыс істеуінен бұрын бірнеше минут күтіңіз.

E
Wi-Fi немесе Ethernet қосылымдарына арналған жабдық жоқ немесе танылмады.
・ Егер желі интерфейсі компьютерге тағылған немесе кірістірілген болса, құрылғының істеп тұрғанын тексеріңіз.
・ Егер желі интерфейсі компьютерге тағылмаған немесе кірістірілмеген болса, принтерді желі арқылы пайдалана алмайсыз. «USB сымды қосылымы» опциясын таңдаңыз да, принтер мен компьютеріңізді USB кабелі арқылы жалғаңыз. Егер принтерді Wi-Fi немесе Ethernet желісі арқылы пайдаланғыңыз келсе, алдымен компьютеріңізге желі орнатыңыз, одан кейін желі қосылымын өзгертіңіз. (Толығырақ Желі нұсқаулығын қараңыз.)
Егер компьютеріңізді немесе желі интерфейсін қалай орнату керектігін білмесеңіз, компьютеріңіз немесе желі интерфейсі құжаттарын қараңыз.
