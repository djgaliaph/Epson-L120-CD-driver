กด บนเครื่องพิมพ์์ค้างไว้ 3 วินาทีภายใน 5 นาที
ไฟสถานะ Wi-Fi กะพริบสลับกัน อาจต้องใช้เวลาสองสามนาทีในการเชื่อมต่อเครื่องพิมพ์เข้ากับเครือข่าย
การตั้งค่า Wi-Fi ล้มเหลวหากไฟดวงซ้ายดับและไฟดวงขวากะพริบอยู่ คลิกที่ลิงค์ต่อไปนี้เพื่อแก้ไขปัญหานี้
การตั้งค่าล้มเหลว
หากหน้าจอ Set Network Location (ตั้งค่าที่ตั้งเครือข่าย) ปรากฏขึ้นบนคอมพิวเตอร์ ให้กด Cancel (ยกเลิก) เพื่อปิดหน้าจอ
แสดงตัวอย่าง "หน้าจอ Set Network Location (ตั้งค่าที่ตั้งเครือข่าย)"
