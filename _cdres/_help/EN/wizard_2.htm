Make a note of your network name (SSID) as shown at the bottom of this screen. Then select this on the printer's screen.
Click the Display Password button at the bottom of this screen. Then enter the password on the printer's screen using the buttons or arrow keys.
NOTE:
The password is case sensitive. Your password may contain uppercase and lowercase letters as well as numbers.
If the password is not displayed even after clicking the button, see the documentation for your wireless router (access point).
