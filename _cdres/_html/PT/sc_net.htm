Informações sobre Ligação à Internet

O software EPSON pode ligar-se à Internet com os seguintes objectivos:
Efectuar actualizações para a versão mais recente
Instalar as funções mais recentes
Este software permite que as funções e funcionalidades mais recentes sejam instaladas a partir do momento em que ficam disponíveis.
Faça clique nas caixas que se seguem para activar ou desactivar estas opções.
Se aparecerem mensagens sobre Firewall durante a instalação, seleccione [Desbloquear] ou [Permitir].
