網路界面的解決方式

問題總覽：

如果您要透過網路連線印表機和電腦，則電腦必須事先連線至網路。 如果您的電腦上沒有連線至區域網路 (Wi-Fi 或乙太網路) 的網路界面，則會顯示〝發生界面錯誤〞畫面。 請參照下列步驟以解決此問題。

步驟 1

首先，參照下列步驟檢查電腦的網路狀態。
Windows 8：選擇〝桌面〞 > 〝設定〞快速鍵 > 〝控制台〞 > 〝檢視網路狀態和工作〞。
Windows 7/Vista：選擇〝開始〞>〝控制台〞>〝檢視網路狀態及工作〞。
Windows XP：選擇〝開始〞>〝控制台〞>〝網路和網際網路連線〞。
Windows 8/7/Vista：點選〝變更介面卡設定〞 (8/7) 或〝管理網路連線〞 (Vista)。
Windows XP：選擇〝開始〞>〝控制台〞>〝網路和網際網路連線〞。
確認已顯示網路連線圖示。

步驟 2

執行適當的解決方法。
解決方法類型
狀態
解決方法

A
電腦具有可進行 Wi-Fi 網路連線的硬體，而且已經啟用，但是沒有連線至網路。

Windows：電腦具有進行 Wi-Fi 網路連線的硬體，但是沒有啟用按鈕或開啟電腦。
・ 如果 Wi-Fi 網路已設定使用無線基地台 (AP)，請檢查該網路裝置 (例如路由器或無線基地台) 已經開啟。 開啟網路裝置後，您需要等候幾分鐘的時間，網路才能正確運作。
・ 如果您沒有設定使用無線基地台 (AP) 的 Wi-Fi 網路，則可以使用 Ad Hoc 模式來無線連線印表機和電腦。如果印表機有〝Wi-Fi Direct〞功能，也可以使用〝Wi-Fi Direct Simple AP〞模式。如需進行 Ad Hoc 模式連線或 Simple AP 模式連線的詳細說明，請參閱〝網路使用說明〞中的〝電腦上的 Ad Hoc 設定〞(Ad Hoc 模式) 或〝Wi-Fi Direct〞(Simple AP 模式)。
・ Windows：啟用網路介面。啟用網路介面後，您需要等候幾分鐘的時間，網路才能正確運作。

B
電腦具有進行 Wi-Fi 網路連線的硬體，但是沒有啟用。
・ 啟用網路狀態。 啟用網路狀態後，您需要等候幾分鐘的時間，網路才能正確運作。

C
電腦具有進行乙太網路連線的硬體，但是沒有連線至網路。
・ 如果已經設定乙太網路，請檢查路由器或集線器已經開啟。
・ 檢查區域網路線已正確連接至路由器、集線器以及電腦。

D
電腦具有進行乙太網路連線的硬體，但是沒有啟用。
・ 啟用網路狀態。 啟用網路狀態後，您需要等候幾分鐘的時間，網路才能正確運作。

E
Wi-Fi 或乙太網路連線的硬體不存在或無法辨識。
・ 如果網路界面已插入電腦或為電腦內建，請確認裝置正在運作中。
・ 如果網路界面沒有插入電腦或電腦並無內建，則您無法透過網路使用印表機。 選擇〝USB 有線連線〞，並透過 USB 線連接印表機和電腦。 如果您要透過 Wi-Fi 或乙太網路使用印表機，首先要設定電腦的網路，然後變更網路連線。 (詳細說明，請參閱「網路使用說明」。)
如果您不知道如何設定電腦或網路界面，請參閱電腦或網路界面的使用手冊。
