اتبع الخطوات التالية.
حدد Start (ابدأ) > Control Panel (لوحة التحكم) > Network and Internet Connections (اتصالات شبكة الاتصال وإنترنت) > Network Connections (اتصالات الشبكة).
انقر بالزر الأيمن للماوس فوق رمز Wireless Network Connection (اتصال الشبكة اللاسلكية) ثم حدد View Available Networks (عرض الشبكات المتوفرة).
حدد SSID المعروض على شاشة LCD الخاصة بالطابعة.
