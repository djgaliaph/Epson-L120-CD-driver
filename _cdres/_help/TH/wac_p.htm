ดำเนินการต่อไปนี้บนแผงควบคุมภายใน 5 นาที
ตัวเลือกเมนูจะแตกต่างกันไปตามรุ่นของเครื่องพิมพ์
สำหรับรุ่นทัชสกรีน ให้กดไอคอนบนหน้าจอของเครื่องพิมพ์ได้โดยตรง
สำหรับรุ่นที่ไม่ใช่ทัชสกรีน ให้ใช้ปุ่มลูกศรเลือกรายการ จากนั้นกด OK (ตกลง)
ตรวจสอบให้แน่ใจว่าเครื่องพิมพ์เปิดเครื่องอยู่และพร้อมทำงาน
ข้อมูลเพิ่มเติม
ไปที่เครื่องพิมพ์และป้อนเมนูการตั้งค่า Wi-Fi โดยทำตามวิธีใดวิธีหนึ่งต่อไปนี้ ทั้งนี้ขึ้นอยู่กับเครื่องพิมพ์ของท่าน
เลือก
เลือก Wi-Fi Setup (การตั้งค่า Wi-Fi)
แตะ ตามด้วย Wi-Fi Setup (การตั้งค่า Wi-Fi)
เลือก Wi-Fi Auto Connect (เชื่อมต่อ Wi-Fi อัตโนมัติ) และกด Proceed (ดำเนินการ) หรือ OK (ตกลง) จากนั้นปฏิบัติตามคำแนะนำบนหน้าจอของเครื่องพิมพ์
ข้อควรทราบ:
หากหน้าจอแสดงข้อผิดพลาดปรากฏขึ้น ให้กด OK (ตกลง) บนแผงควบคุมของเครื่องพิมพ์ จากนั้นกลับไปสู่ขั้นตอนที่ 1 บนหน้าจอนี้
หากหน้าจอ Set Network Location (ตั้งค่าที่ตั้งเครือข่าย) ปรากฏขึ้นบนคอมพิวเตอร์ ให้กด Cancel (ยกเลิก) เพื่อปิดหน้าจอ
แสดงตัวอย่าง "หน้าจอ Set Network Location (ตั้งค่าที่ตั้งเครือข่าย)"
