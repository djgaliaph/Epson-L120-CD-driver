Führen Sie die folgende Bedienung am Bedienfeld innerhalb von sieben Minuten aus.
Die Menüauswahl variiert je nach Druckermodell.
Drücken Sie bei Modellen mit Touchscreen das Symbol direkt auf dem Druckerbildschirm.
Verwenden Sie bei Modellen ohne Touchscreen zur Auswahl der Menüpunkte die Pfeiltasten. Drücken Sie danach OK.
Stellen Sie sicher, dass der Drucker eingeschaltet und betriebsbereit ist.
Weitere Informationen
Rufen Sie am Drucker mit einer der folgenden Methoden, je nach Drucker, das Wi-Fi-Setup-Menü auf.
Wählen Sie .
Wählen Sie Wi-Fi-Setup.
Tippen Sie auf und anschließend auf Wi-Fi-Setup.
Wählen Sie Autom. WLAN-Verbindung und drücken Sie Fortfahren oder OK. Folgen Sie dann der Anleitung im Druckerbildschirm.
HINWEIS:
Wenn ein Fehlerbildschirm angezeigt wird, drücken Sie OK im Druckerbedienfeld. Führen Sie dann Schritt 1 in diesem Bildschirm erneut aus.
Wenn der Bildschirm Set Network Location (Netzwerkstandort einrichten) auf dem Computer angezeigt wird, drücken Sie Abbrech., um den Bildschirm zu schließen.
