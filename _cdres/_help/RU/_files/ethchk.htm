Поиск и устранение неисправностей для Отчет о сетевых подключениях

Для подключения Wi-Fi

Подключение Ethernet

Порядок определения статуса сети и решения проблем с подключением

  1. Просмотрите распечатанный отчет о сетевом подключении. Проверить пункты в A (Проверить соединение сети), которые имеют статус НЕ ПРОЙДЕНО.
  2. В списке пунктов проверки выбрать элементы, которые имеют статус НЕ ПРОЙДЕНО, чтобы отобразить способы устранения проблемы.
  3. Проверить сообщение, отображаемое в B, и отыскать такое же сообщение, из перечисленных в каждом списке проверок. Затем выполнить способ устранения проблемы.


Пункты проверки

Выберите следующие пункты для отображения сообщений и способов устранения проблемы.
  1. Ethernet Cable Connection Check (Проверить соединение кабеля Ethernet)
  2. IP Address Check (Проверить IP-адрес)
  3. Detailed IP Setup Check (Проверить подробную установку IP)


Сообщения и способы устранения проблемы для пунктов проверки

1. Ethernet Cable Connection Check (Проверить соединение кабеля Ethernet)

№ Сообщения Способы устранения
1 Подтвердить, что сетевой кабель подключен и такие сетевые устройства, как концентратор, маршрутизатор или точка доступа включены. Проверить следующее:
  • Сетевой кабель подключен к таким сетевым устройствам, как принтер или концентратор
  • Сетевые устройства, такие как концентратор, включены
Проверить, что сетевой кабель подключен и работает правильно.
Рабочее состояние сетевого устройства можно определить по состоянию индикатора сетевого порта на подключенном устройстве, таком как концентратор.

2. IP Address Check (Проверить IP-адрес)

№ Сообщения Способы устранения
2 Принтеру присвоен неправильный IP-адрес. Подтвердить установку IP-адреса сетевого устройства (концентратор, маршрутизатор или точка доступа). Свяжитесь с системным администратором для поддержки. Проверить, включена ли функция DHCP для точки доступа. Если она выключена, включить ее.

3. Detailed IP Setup Check (Проверить подробную установку IP)

№ Сообщения Способы устранения
3 Подтвердить установку IP-адреса, маски подсети и шлюза по умолчанию.
Доступно соединение с использованием установки EpsonNet.
Свяжитесь с системным администратором для поддержки.
  1. Проверить следующее:
    • Другие устройства в сети включены
    • Сетевой кабель подключен верно
    • Установленный вручную сетевой адрес принтера верный
    • Сетевой адрес принтера совпадает с адресом одного из устройств
  2. Если сетевой адрес неверный, установить верный адрес с помощью ЖК-панели принтера.
    Если невозможно изменить параметры с помощью принтера, установите их с помощью EpsonNet Setup.
    Текущий IP-адрес, маска подсети и шлюз по умолчанию отображаются в C (Network Status (Статус сети)).
4 Установка не завершена. Подтвердить установку шлюза по умолчанию.
Доступно соединение с использованием установки EpsonNet.
Свяжитесь с системным администратором для поддержки.
  1. Проверить следующее:
    • Установленный вручную для принтера адрес шлюза по умолчанию верный
    • Устройство, указанное как шлюз по умолчанию, включено
    • Сетевой кабель подключен верно
  2. Если адрес шлюза по умолчанию неверный, установить верный адрес с помощью ЖК-панели принтера.
    Если невозможно изменить параметры с помощью принтера, установите их с помощью EpsonNet Setup.
    Текущий адрес шлюза по умолчанию отображается в C (Network Status (Статус сети)) в поле Default Gateway.
5 Подтвердить соединение и настройки сети ПК или другого устройства.
Доступно соединение с использованием установки EpsonNet.
Свяжитесь с системным администратором для поддержки.
  1. Проверить следующее:
    • Другие устройства в сети включены
    • Сетевой кабель подключен верно
    • Установленный вручную сетевой адрес принтера верный
    • Сетевой адрес принтера совпадает с адресом одного из устройств
  2. Если все указанные выше пункты проверены, выполнить следующее:
    • Запустить EpsonNet Setup на ПК, который находится в одной подсети с принтером. Если принтер указан в списке принтеров, установить IP-адрес принтера на экране настройки IP-адреса.
Проверить IP-адрес принтера и проверить, доступен ли сетевой принтер для других устройств в сети. В некоторый случаях ошибка может здесь не указываться. Рекомендуется выполнять настройку сети с помощью EpsonNet Setup.
