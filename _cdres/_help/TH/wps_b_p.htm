ตัวเลือกเมนูจะแตกต่างกันไปตามรุ่นของเครื่องพิมพ์
สำหรับรุ่นทัชสกรีน ให้กดไอคอนบนหน้าจอของเครื่องพิมพ์ได้โดยตรง
สำหรับรุ่นที่ไม่ใช่ทัชสกรีน ให้ใช้ปุ่มลูกศรเลือกรายการ จากนั้นกด OK (ตกลง)
ตรวจสอบให้แน่ใจว่าเครื่องพิมพ์เปิดเครื่องอยู่และพร้อมทำงาน
ข้อมูลเพิ่มเติม
ไปที่เครื่องพิมพ์และป้อนเมนูการตั้งค่า Wi-Fi โดยทำตามวิธีใดวิธีหนึ่งต่อไปนี้ ทั้งนี้ขึ้นอยู่กับเครื่องพิมพ์ของท่าน
เลือก
เลือก Wi-Fi Setup (การตั้งค่า Wi-Fi)
แตะ ตามด้วย Wi-Fi Setup (การตั้งค่า Wi-Fi)
เลือก Push Button Setup (WPS) (การตั้งค่าปุ่มกด (WPS))/ Push Button (WPS) (ปุ่มกด (WPS))
กดปุ่ม WPS บนเราเตอร์ไร้สาย (จุดเข้าใช้งานแบบไร้สาย) จากนั้นกด Proceed (ดำเนินการ) หรือ OK (ตกลง) บนเครื่องพิมพ์
ข้อควรทราบ:
หากเราเตอร์ไร้สาย (จุดเข้าใช้งานแบบไร้สาย) ไม่มีปุ่ม ให้เปิดใช้ WPS โดยใช้ซอฟต์แวร์ที่ให้มากับเราเตอร์ไร้สาย (จุดเข้าใช้งานแบบไร้สาย)
เมื่อข้อความว่าเสร็จเรียบร้อยแล้วปรากฏขึ้น ให้กด OK (ตกลง) เพื่อสิ้นสุดการตั้งค่า Wi-Fi
คลิกลิงค์ต่อไปนี้ หากการตั้งค่า Wi-Fi ล้มเหลว
การตั้งค่าล้มเหลว
