Wykonaj poniższe czynności.
Wybierz Start > Control Panel (Panel sterowania) > Network and Internet Connections (Sieć i połączenia internetowe) > Network Connections (Połączenia sieciowe).
Kliknij prawym przyciskiem myszy ikonę Wireless Network Connection (Połączenie sieci bezprzewodowej), a następnie wybierz opcję View Available Networks (Wyświetl dostępne sieci).
Wybierz identyfikator SSID wyświetlony na ekranie LCD drukarki.
