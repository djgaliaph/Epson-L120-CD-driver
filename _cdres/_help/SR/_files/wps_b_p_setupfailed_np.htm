Press to clear the error, check the following items, and then perform network setup again.
Check that the wireless router (the access point) is turned on. If it is turned off, turn it on and then perform network setup again.
If you performed a PIN code seup, check that the PIN code entered in the wireless router (the access point) utility is correct.
Place your printer close to the wireless router (the access point).
Check that the Wi-Fi communication mode (IEEE802.11*) set for your wireless router (the access point) matches the one supported by your printer. (*Display varies according to your location.)
If the MAC address filtering function is enabled, register your printer's MAC address to prevent your printer from being filtered.
Check whether or not the DHCP function for the wireless router (the access point) is enabled. If it is disabled, change it to enabled.
