Aşağıdaki adımları izleyin.
Sırasıyla, Start (Başlat) > Control Panel (Denetim Masası) > Network and Internet Connections (Ağ ve İnternet Bağlantıları) > Network Connections (Ağ Bağlantıları) öğelerini seçin.
Wireless Network Connection (Kablosuz Ağ Bağlantısı) simgesine sağ tıklayın ve ardından View Available Networks (Kullanılabilir Ağları Görüntüle) öğesini seçin.
Yazıcının LCD ekranında görüntülenen SSID'ye tıklayın.
