ตัวเลือกเมนูจะแตกต่างกันไปตามรุ่นของเครื่องพิมพ์
สำหรับรุ่นทัชสกรีน ให้กดไอคอนบนหน้าจอของเครื่องพิมพ์ได้โดยตรง
สำหรับรุ่นที่ไม่ใช่ทัชสกรีน ให้ใช้ปุ่มลูกศรเลือกรายการ จากนั้นกด OK (ตกลง)
ตรวจสอบให้แน่ใจว่าเครื่องพิมพ์เปิดเครื่องอยู่และพร้อมทำงาน
ข้อมูลเพิ่มเติม
ไปที่เครื่องพิมพ์และป้อนเมนูการตั้งค่า Wi-Fi โดยทำตามวิธีใดวิธีหนึ่งต่อไปนี้ ทั้งนี้ขึ้นอยู่กับเครื่องพิมพ์ของท่าน

เลือก Wi-Fi Direct Routerless Setup (การตั้งค่า Wi-Fi Direct แบบไม่มีเราเตอร์)/ Wi-Fi Direct Setup (การตั้งค่า Wi-Fi Direct)
เลือก Wi-Fi Direct Routerless Setup (การตั้งค่า Wi-Fi Direct แบบไม่มีเราเตอร์)/ Wi-Fi Direct Setup (การตั้งค่า Wi-Fi Direct) ตามด้วย Connection Setup (การตั้งค่าการเชื่อมต่อ)
ปฏิบัติตามคำแนะนำบนหน้าจอของเครื่องพิมพ์ แล้วกด Proceed (ดำเนินการ) หรือ OK (ตกลง)
ตรวจสอบ SSID ที่ปรากฏขึ้นบนหน้าจอของเครื่องพิมพ์
หากไม่ปรากฏขึ้น ให้กด OK (ตกลง) หรือ แล้วค้นหาและเลือก SSID ด้วยคอมพิวเตอร์
