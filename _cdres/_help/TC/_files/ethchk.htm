網路連線報告的解決方法

關於 Wi-Fi 連線

乙太網路連線

如何解決網路狀態或連線問題

  1. 查看列印出的〝網路連線報告〞。檢查在 A 〝Check Network Connection (檢查網路連線)〞中狀態顯示為〝FAIL (失敗)〞的項目。
  2. 在〝檢查項目〞清單中，點選狀態顯示為〝FAIL (失敗)〞的項目即可顯示解決方法。
  3. 檢查在 B 中所顯示的訊息，並尋找每個〝檢查項目〞下列出的相同訊息。然後請參照解決方法。


檢查項目

點選下列項目，顯示訊息和解決方法。
  1. Ethernet Cable Connection Check (乙太網路線連接檢查)
  2. IP Address Check (IP 位址檢查)
  3. Detailed IP Setup Check (詳細的 IP 設定檢查)


檢查項目的訊息與解決方法

1. Ethernet Cable Connection Check (乙太網路線連接檢查)

編號 訊息 解決方法
1 Confirm that the network cable is connected and network devices such as hub, router, or access point are turned on. 檢查下列項目：
  • 已將網路線連接到印表機或集線器等網路裝置
  • 已將集線器等網路裝置的電源開啟
檢查是否已連接網路線且正確運作。
您可以檢查集線器等連線裝置上網路連接埠的狀態 LED，以確認網路裝置的運作狀態。

2. IP Address Check (IP 位址檢查)

編號 訊息 解決方法
2 Incorrect IP address is assigned to the printer. Confirm IP address setup of the network device (hub, router, or access point). Contact your network administrator for assistance. 檢查是否已開啟無線基地台 (AP) 的 DHCP 功能。若沒有開啟，請開啟此功能。

3. Detailed IP Setup Check (詳細的 IP 設定檢查)

編號 訊息 解決方法
3 Confirm IP address, subnet mask, and default gateway setup.
Connection using the EpsonNet Setup is available.
Contact your network administrator for assistance.
  1. 檢查下列項目：
    • 已開啟網路上其他裝置的電源。
    • 已正確連接網路線。
    • 手動為印表機設定的網路位址正確
    • 印表機的網路位址與其他裝置的網路位址相同
  2. 如果網路位址不正確，請使用印表機的液晶面板設定正確的位址。
    如果您無法使用印表機來變更設定，請利用 EpsonNet Setup 來進行設定。
    您目前的 IP 位址、子網路遮罩、和預設閘道會顯示在 C 〝Network Status (網路狀態)〞中。
4 Setup is incomplete. Confirm default gateway setup.
Connection using the EpsonNet Setup is available.
Contact your network administrator for assistance.
  1. 檢查下列項目：
    • 手動為印表機設定的預設閘道位址正確
    • 已開啟指定為預設閘道的裝置電源。
    • 已正確連接網路線。
  2. 如果預設閘道位址不正確，請使用印表機的液晶面板設定正確的位址。
    如果您無法使用印表機來變更設定，請利用 EpsonNet Setup 來進行設定。
    您目前的預設閘道位址會顯示在 C 〝Network Status (網路狀態)〞的〝Default Gateway (預設閘道)〞中。
5 Confirm the connection and network setup of the PC or other device.
Connection using the EpsonNet Setup is available.
Contact your network administrator for assistance.
  1. 檢查下列項目：
    • 已開啟網路上其他裝置的電源。
    • 已正確連接網路線。
    • 手動為印表機設定的網路位址正確
    • 印表機的網路位址與其他裝置的網路位址相同
  2. 如果您已檢查上述各點，請試試下列項目：
    • 在與印表機使用相同網路的電腦上啟動 EpsonNet Setup。如果您的印表機列在印表機清單中，請在 IP 位址設定畫面上設定印表機的 IP 位址。
檢查印表機的 IP 位址，並檢查網路印表機是否能夠存取網路上的其他裝置。在某些情況下，可能會無法指出錯誤。我們建議您使用 EpsonNet Setup 執行網路設定。
