Välj Start > Control Panel (Kontrollpanelen) > Network and Internet Connections (Nätverks- och Internet-inställningar) > Network Connections (Nätverksanslutningar).
Högerklicka på ikonen Wireless Network Connection (Trådlös nätverksanslutning) och välj View Available Networks (Visa tillgängliga nätverk).
Välj det SSID som visas på skrivarens LCD-skärm.
