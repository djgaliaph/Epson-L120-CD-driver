Wi-Fi қосылымы үшін

Ethernet қосылымы

Желі күйінің немесе қосылым ақауларын шешу әдісі

  1. Басып шығарылған Желі қосылымы есебін қараңыз. A ішіндегі элементтерді тексеру Күйдің FAIL (СӘТСІЗ) деп көрсеткенін (Check Network Connection (Желі қосылымын тексеру)).
  2. Элементтерді тексеру тізімінде, шешімдерді көрсету үшін FAIL (СӘТСІЗ) күйін көрсететін элементтерді басыңыз.
  3. B көрсетілген хабарды тексеріңіз және әрбір элементті тексеру тізіміндегі бірдей хабарды іздеңіз. Одан кейін, шешімдерді орындаңыз.


Элементтерді тексеру

Хабарлар мен шешімдерге өту үшін төмендегі элементтерді басыңыз.
  1. Ethernet Cable Connection Check (Ethernet кабелі қосылымын тексеру)
  2. IP Address Check (IP мекенжайын тексеру)
  3. Detailed IP Setup Check (Толық IP орнатуды тексеру)


Элементтерді тексеру хабарлары мен шешімдері

1. Ethernet Cable Connection Check (Ethernet кабелі қосылымын тексеру)

№ Хабарлар Шешімдер
1 Confirm that the network cable is connected and network devices such as hub, router, or access point are turned on (Желі кабелі жалғанғанын және хаб, жол жоспарлағыш немесе кіру нүктесі сияқты құрылғылар қосылғанын растаңыз). Төмендегіні тексеріңіз:
  • желі кабелі принтер немесе хаб сияқты желі құрылғысына жалғанған
  • хаб сияқты желі құрылғысы қосылған
Желі кабелінің жалғанғанын және дұрыс істеп тұрғанын тексеріңіз.
Желі құрылғыларының жұмыс күйін хаб сияқты қосылған құрылғылардың желі портындағы жарық диодты шамның күйін тексеру арқылы растауға болады.

Жоғарғы

2. IP Address Check (IP мекенжайын тексеру)

№ Хабарлар Шешімдер
2 Incorrect IP address is assigned to the printer (Принтерге дұрыс емес IP мекенжайы тағайындалған). Confirm IP address setup of the network device (hub, router, or access point) (Желі құрылғысына (хаб, жол жоспарлағыш немесе кіру нүктесі) IP мекенжайы орнатылғанын тексеріңіз). Contact your network administrator for assistance (Көмек алу үшін желі әкімшісіне хабарласыңыз). Кіру нүктесінің DHCP функциясы қосылғанын тексеріңіз. Егер ажыратылған болса, оны қосыңыз.

3. Detailed IP Setup Check (Толық IP орнатуды тексеру)

№ Хабарлар Шешімдер
3 Confirm IP address, subnet mask, and default gateway setup (IP мекенжайы, ішкі желі пердесі және әдепкі шлюз орнатылғанын растаңыз).
Connection using the EpsonNet Setup is available (EpsonNet Setup арқылы қосылу қол жетімді).
Contact your network administrator for assistance (Көмек алу үшін желі әкімшісіне хабарласыңыз).
  1. Төмендегіні тексеріңіз:
    • желідегі басқа құрылғылар қосылғанын
    • желі кабелі дұрыс жалғанғанын
    • принтерге қолмен орнатылған желі мекенжайы дұрыс екенін
    • принтердің желі мекенжайы басқа құрылғылармен бірдей екенін
  2. Егер желі мекенжайы дұрыс болмаса, принтердің СК панелі арқылы дұрыс мекенжай орнатыңыз.
    Егер принтер арқылы параметрлерді өзгерте алмасаңыз, параметрлерді EpsonNet Setup арқылы орнатыңыз.
    Ағымдағы IP мекенжайы, ішкі желі пердесі және әдепкі шлюз C (Network Status (Желі күйі)) көрсетілген.
4 Setup is incomplete (Орнату аяқталмаған). Confirm default gateway setup (Әдепкі шлюз орнатуды растаңыз).
Connection using the EpsonNet Setup is available (EpsonNet Setup арқылы қосылу қол жетімді).
Contact your network administrator for assistance (Көмек алу үшін желі әкімшісіне хабарласыңыз).
  1. Төмендегіні тексеріңіз:
    • принтерге қолмен орнатылған әдепкі шлюз мекенжайы дұрыс екенін
    • әдепкі шлюз ретінде көрсетілген құрылғы қосылғанын
    • желі кабелі дұрыс жалғанғанын
  2. Егер әдепкі шлюз мекенжайы дұрыс болмаса, принтердің СК панелі арқылы дұрыс мекенжай орнатыңыз.
    Егер принтер арқылы параметрлерді өзгерте алмасаңыз, параметрлерді EpsonNet Setup арқылы орнатыңыз.
    Ағымдағы әдепкі шлюз мекенжайыңыз Әдепкі шлюз астындағы C-де (Network Status (Желі күйі)) көрсетіледі.
5 Confirm the connection and network setup of the PC or other device (Компьютердің немесе басқа құрылғының қосылымын және желі қосылымын растаңыз).
Connection using the EpsonNet Setup is available (EpsonNet Setup арқылы қосылу қол жетімді).
Contact your network administrator for assistance (Көмек алу үшін желі әкімшісіне хабарласыңыз).
  1. Төмендегіні тексеріңіз:
    • желідегі басқа құрылғылар қосылғанын
    • желі кабелі дұрыс жалғанғанын
    • принтерге қолмен орнатылған желі мекенжайы дұрыс екенін
    • принтердің желі мекенжайы басқа құрылғылармен бірдей екенін
  2. Егер жоғарыдағы барлық пункттерді тексерсеңіз, төмендегіні байқап көріңіз:
    • Принтер сияқты бірдей желіні пайдаланатын компьютердегі EpsonNet Setup қолданбасын іске қосыңыз. Егер принтеріңіз принтер тізімінде болса, принтердің IP мекенжайын IP мекенжайын орнату экранында орнатыңыз.
Принтердің IP мекенжайын тексеріңіз және желі принтері желідегі басқа құрылғыларға кіре алатынын тексеріңіз. Кейбір жағдайда, қате көрсетілмеуі мүмкін. Желі орнатуды EpsonNet Setup арқылы орындауды ұсынамыз.
