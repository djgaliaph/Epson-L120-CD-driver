Нажмите кнопку WPS на беспроводном маршрутизаторе (точке доступа).
ПРИМЕЧАНИЕ.
Если на беспроводном маршрутизаторе (точке доступа) нет кнопки, активируйте WPS с помощью программного обеспечения из комплекта беспроводного маршрутизатора (точки доступа).
Удерживайте нажатой кнопку на принтере в течение трех секунд.
Индикаторы состояния Wi-Fi мигают попеременно. Для подключения принтера к сети может понадобиться несколько минут.
Когда оба индикатора состояния сети включаются, как показано на следующем рисунке, настройка Wi-Fi завершена. Правый индикатор состояния сети выключается через пять минут.
Если левый индикатор не светится, а правый мигает, настройку Wi-Fi выполнить не удалось. Перейдите по следующей ссылке для решения проблемы.
