网络连接检查报告的故障排除

对于 Wi-Fi 连接

以太网连接

如何解决网络状态或连接问题

  1. 查看您所打印的网络连接检查报告。 检查 A（Check Network Connection（检查网络连接））中状态显示为“FAIL（失败）”的项目。
  2. 在“检查项目”列表中，单击状态显示为“FAIL（失败）”的项目，显示其解决方法。
  3. 检查 B 中显示的信息，并在各个“检查项目”下查找列出的相同信息。 然后按照解决方法进行操作。


检查项目

单击下方的项目，查看信息和解决方法。
  1. Ethernet Cable Connection Check（检查以太网线缆连接）
  2. IP Address Check（检查IP地址）
  3. Detailed IP Setup Check（检查详细的IP设置）


检查项目的信息和解决方法

1. Ethernet Cable Connection Check（检查以太网线缆连接）

编号 信息 解决方法
1 Confirm that the network cable is connected and network devices such as hub, router, or access point are turned on.（请确认网络线缆已连接，并且集线器、路由器或接入点等网络设备均已开启。） 检查下列项目：
  • 网络线缆已连接到打印机或集线器等网络设备
  • 集线器等网络设备已开启
检查网络线缆是否已连接并可正常工作。
您可通过检查集线器等所连设备上网络端口的 LED 指示灯状态，来确认网络设备的运行状态。

2. IP Address Check（检查IP地址）

编号 信息 解决方法
2 Incorrect IP address is assigned to the printer.（打印机分配了不正确的 IP 地址。)Confirm IP address setup of the network device (hub, router, or access point).（请确认网络设备（集线器、路由器或接入点）的 IP 地址设置。） Contact your network administrator for assistance.（请联系您的网络管理员以获得帮助。） 检查是否已启用接入点的 DHCP 功能。 若未启用，将它改为启用。

3. Detailed IP Setup Check（检查详细的IP设置）

编号 信息 解决方法
3 Confirm IP address, subnet mask, and default gateway setup.（请确认 IP 地址、子网掩码和默认网关设置。）
Connection using the EpsonNet Setup is available.（可通过 EpsonNet Setup 进行连接。）
Contact your network administrator for assistance.（请联系您的网络管理员以获得帮助。）
  1. 检查下列项目：
    • 已开启网络上的其他设备
    • 已正确连接网络线缆
    • 为打印机手动设置的网络地址是正确的
    • 打印机的网络地址与其他设备的相同
  2. 如果网络地址不正确，请通过打印机的液晶显示屏设置正确的地址。
    如果无法通过打印机更改设置，请使用EpsonNet Setup进行设置。
    在C（Network Status（网络状态））中显示当前的IP地址、子网掩码和默认网关。
4 Setup is incomplete.（设置不完整。）Confirm default gateway setup.（请确认默认网关设置。）
Connection using the EpsonNet Setup is available.（可通过 EpsonNet Setup 进行连接。）
Contact your network administrator for assistance.（请联系您的网络管理员以获得帮助。）
  1. 检查下列项目：
    • 为打印机手动设置的默认网关地址是正确的
    • 指定为默认网关的设备已开启
    • 已正确连接网络线缆
  2. 如果默认网关地址不正确，请通过打印机的液晶显示屏设置正确的地址。
    如果无法通过打印机更改设置，请使用 EpsonNet Setup 进行设置。
    在C（Network Status（网络状态））中“默认网关”下显示当前的默认网关地址。
5 Confirm the connection and network setup of the PC or other device.（请确认计算机或其他设备的连接和网络设置。）
Connection using the EpsonNet Setup is available.（可通过 EpsonNet Setup 进行连接。）
Contact your network administrator for assistance.（请联系您的网络管理员以获得帮助。）

    • 打印机的网络地址与其他设备的相同

    • 在使用与打印机相同网络的计算机上，启动 EpsonNet Setup。 如果打印机列表中列出了您的打印机，请在IP 地址设置屏幕中设置该打印机的IP地址。
检查打印机的 IP 地址，并查看网络打印机是否可以访问网络上的其他设备。 在某些情形中，错误无法认定。 建议您使用 EpsonNet Setup 进行网络设置。
