Atlikite toliau nurodytus veiksmus.
Pasirinkite Start (pradžia) > Control Panel (valdymo skydelis) > Network and Internet Connections (tinklo ir interneto ryšiai) > Network Connections (tinklo ryšiai).
Dešiniuoju pelės klavišu spustelėkite piktogramą Wireless Network Connection (belaidžio tinklo ryšys) ir pasirinkite View Available Networks (žiūrėti pasiekiamus tinklus).
Pasirinkite spausdintuvo skystųjų kristalų ekrane rodomą SSID.
