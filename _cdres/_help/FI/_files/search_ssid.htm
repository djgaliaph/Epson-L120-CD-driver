Noudata alla olevia ohjeita.
Valitse Start (Käynnistä) > Control Panel (Ohjauspaneeli) > Network and Internet Connections (Verkko- ja Internet-yhteydet) > Network Connections (Verkkoyhteydet).
Napsauta hiiren kakkospainikkeella Wireless Network Connection (Langaton verkkoyhteys) -kuvaketta ja valitse sitten View Available Networks (Näytä käytettävissä olevat verkot).
Valitse tulostimen LCD-näytössä näkyvä SSID-tunnus.
